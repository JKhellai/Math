# 数学知识体系 · Mappa Madre della Matematica

一张可下钻的 **3D 数学知识图**：我从最底层的逻辑与集合出发，一砖一瓦地重构属于自己的数学知识体系，并把这条重构之路画成一张会**慢慢生长**的地图。

🔗 **在线查看**：<https://JKhellai.github.io/Math/>

---

## 这是什么

这是一个**个人项目**。它对我自己的意义其实也还在生长中——姑且说：**为了从基础重新整理、重构我的数学知识体系**。每当我把一个概念真正想清楚、把它接回赖以成立的根基，就把它作为一个节点加进来。所以这张图会**随着我的重构进度逐步更新**，而不是一次画完的成品。

图里有两套关系：

- **归属**（一个概念属于哪一块领域）—— 用**位置**与**下钻层级**表达；
- **依赖**（一个概念建立在哪些更基本的概念之上）—— 用**连线**表达。这是全图的骨架：*没有无根之木*，每条依赖链最终都落在某条公理上。

节点颜色由三大学科主干（**分析 / 几何 / 代数**）按依赖来源在 OKLab 色彩空间融合而成；**地基**（逻辑、集合、关系、函数……）取中性灰，是一切的底座。高度则编码"建立在……之上"——地基在最底层，越往上的概念依赖越深，像一棵从地基长出来的树。

## 怎么看

打开在线链接，或克隆本仓库后**双击 `index.html`**（自包含，无需服务器）。

- **拖拽**旋转视角，**滚轮**缩放；
- **单击**一个节点：高亮它的依赖链——上游（它所依赖的根）金色、下游（依赖它的）青色，其余淡出；右侧滑出该节点的课文；
- **双击**带一圈外环的"容器"节点：**下钻**进入它的子图；
- **悬停**节点、或课文里的 `[[名词]]`：弹出一句话简介；点该词可在面板内顺着前置概念一路读下去；
- 顶栏有**后退 / 前进**（统一的浏览历史栈）、**面包屑**（当前所在位置，可点击跳转）、**搜索**；
- 课文面板可**拖宽**，或一键 **专注阅读**（图缩成角落小地图）。

## 它是怎么搭起来的

```
content/**/*.md   →   build.py   →   index.html（自包含，可双击）
（唯一真相来源）      （解析 + 着色 + 布局）
```

- 每个概念 = 一个带 YAML frontmatter 的 Markdown 文件。**自底向上**：每个文件只声明自己的 `parent`（归属）与 `prerequisites`（依赖）；归属树、依赖网、颜色、坐标、容器/叶身份全部由 `build.py` **推导**得出，不在第二处手存。
- 查看器是单个**自包含 HTML**：数据内嵌其中，第三方库走 CDN（Three.js / marked / KaTeX），因此 `file://` 双击即开。

### 本地构建

Windows 下可直接**双击 `build.bat`**。
