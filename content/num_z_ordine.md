## 两份文书,一处根源

上一节末尾留下两件未办之事:ℤ 自称**整环**(integral domain),证书未发;ℕ 的序要**移植**给 ℤ,让「负数小于正数」获得定义。两件事看似无关,实则同根——都绕不开对 ℤ 做**正负剖分**:哪些元素算「正」,哪些算「负」,剖分清楚了,序与零因子的问题便都有了落点。而剖分的钥匙,上一节练习 (b) 已经交到你手里:每个整数恰是某个 ι(n) 或某个 −ι(n)。

## 先替 ℕ 补一条引理:无零因子

整环证书的最后一块垫脚石,出人意料地落回 ℕ。

**引理(ℕ 无零因子).** 设 m, n ∈ ℕ,m ≠ 0 且 n ≠ 0,则 m·n ≠ 0。

**证明.** 由[[num_ordine|引理「零或后继」]],m = S(p)、n = S(q) 对某 p, q 成立。展开:

m·n = S(p)·S(q) = S(p)·q + S(p) = S( S(p)·q + p ),

第二步是乘法定义 M2,第三步是 A2(x + S(p) = S(x+p))。它是一个后继,而公理 P4 说 0 不是任何数的后继,故 m·n ≠ 0。∎

又是 P4 在把关。这条引理眼下貌不惊人,稍后它将独力撑起 ℤ 的整环身份。

## ℤ 上的序:把 = 放宽成 ≤

如何在类上比大小?直觉照旧先行:[(a,b)] 是「a − b」,[(c,d)] 是「c − d」,「a − b ≤ c − d」移项应为「a + d ≤ c + b」——右边只用 ℕ 的加法与序,全是现成家底。于是:

**定义.** [(a, b)] ≤ [(c, d)] :⟺ a + d ≤ c + b。又 x < y :⟺ x ≤ y ∧ x ≠ y。

把它与等价关系的定义式并排看:**(a,b) ~ (c,d) ⟺ a + d = c + b**。序的定义,不多不少,就是把 ~ 里的 **= 放宽成 ≤**。这不是巧合:等价说「差为零」,序说「差不超过零」——一对孪生定义,接下来你会看到孪生的证明。

但请先慢一步——这又是一个**按代表元**的定义,[[num_z|⊕ 的教训]]同样适用,只是这次轮到**关系**:必须验证「真假不随代表的更换而改变」,否则 ≤ 根本不是 ℤ 上的关系,只是一纸空文。先备一条一行的小引理:**m + k ≤ n + k ⟹ m ≤ n**(取见证 j 使 (m+k)+j = n+k,重排为 (m+j)+k = n+k,加法消去律消去 k,得 m+j = n)。

**定理(序的良定义).** 设 (a,b) ~ (a′,b′),(c,d) ~ (c′,d′),且 a + d ≤ c + b。则 a′ + d′ ≤ c′ + b′。

**证明.** 已知 a + b′ = a′ + b …(E1),c + d′ = c′ + d …(E2)。在不等式 a + d ≤ c + b 两边同加 b′ + d′(序与加法相容,上一节定理):

(a + b′) + (d + d′) ≤ (c + d′) + (b + b′)。

左边用 (E1) 替换:( a′ + b ) + (d + d′) = (a′ + d′) + (b + d);右边用 (E2) 替换:(c′ + d) + (b + b′) = (c′ + b′) + (b + d)。于是

(a′ + d′) + (b + d) ≤ (c′ + b′) + (b + d),

由小引理消去 (b+d),得 a′ + d′ ≤ c′ + b′。∎

## 它是全序

四条性质,条条是 ℕ 对应性质的镜像;其中反对称一条的「落点」最值得玩味。设 x = [(a,b)],y = [(c,d)],z = [(e,f)]。

**自反**:x ≤ x ⟺ a + b ≤ b + a,由交换律即 a+b ≤ a+b,成立。

**传递**:已知 a+d ≤ c+b 与 c+f ≤ e+d。两式相加(不等式可加:x≤y 且 u≤v,则 x+u ≤ y+u ≤ y+v,两次相容加一次传递),重排:

(a + f) + (c + d) ≤ (e + b) + (c + d),

消去 (c+d),得 a + f ≤ e + b,即 x ≤ z。请把它与上一节 ~ 的传递性证明并排放:**同一份证明,= 改作 ≤**——孪生定义,孪生论证。

**反对称**:x ≤ y 且 y ≤ x,即 a+d ≤ c+b 且 c+b ≤ a+d;由 ℕ 的反对称性,a + d = c + b——**这恰是 (a,b) ~ (c,d)**,即 x 与 y 是同一个类。看清落点:在 ℕ,反对称夹出的是相等;在 ℤ,夹出的是等价——而商集世界里,等价**就是**相等。当初「把视为相同升格为真的相同」那句话,在此处兜了一个漂亮的圈。

**全序性**:a+d 与 c+b 都是自然数,ℕ 的全序性断言二者必可比较,无论哪边成立,按定义都给出 x ≤ y 或 y ≤ x。∎

## 正负剖分与数轴的另一半

序既立,问 0 在哪边。0 = [(0,0)],按定义

0 ≤ [(a, b)] ⟺ 0 + b ≤ a + 0 ⟺ b ≤ a,

而上一节练习 (b) 说:b ≤ a 时 [(a,b)] = ι(a−b)。所以**非负整数恰是 ι(ℕ)**——ℕ 的拷贝,一个不多一个不少。对称地,[(a,b)] ≤ 0 ⟺ a ≤ b ⟺ 该类形如 −ι(n)。配上全序与反对称,得 ℤ 的**三歧性**:每个整数,负、零、正,恰居其一。

顺手核验嵌入的最后一项资格:**ι 保序**——m ≤ n ⟺ m + 0 ≤ n + 0 ⟺ ι(m) ≤ ι(n),两个方向同时到手。至此 ι 保加法、保乘法、保序、且单:ℕ 在 ℤ 中的这份拷贝,连大小关系都原样保存。「安家」的全部手续齐了。

## 序与运算的相容

两条相容律,正是 ℕ 那两条的延续:

**与加法**:x ≤ y ⟹ x + z ≤ y + z。(代表元上即「两边同加 e+f 再重排」,一行。)

**与乘法**:0 ≤ x 且 0 ≤ y ⟹ 0 ≤ x·y。(由剖分,x = ι(m)、y = ι(n);则 x·y = ι(m·n) ∈ ι(ℕ),非负。)

「全序 + 这两条相容」的配置有个名字:**有序环**(ordered ring)。它的域版本——有序域——正是 ℚ 与 ℝ 将要佩戴的完整身份;承诺已久的「平方非负」,源头已近在眼前(见练习之后)。

## ℤ 是整环

**定理.** 在 ℤ 中,x·y = 0 ⟹ x = 0 或 y = 0。

**证明.** 证逆否:设 x ≠ 0 且 y ≠ 0,往证 x·y ≠ 0。由三歧性与剖分,x = ι(m) 或 x = −ι(m),其中 m ≠ 0(若 m = 0 则 x = 0);y 同理,y = ±ι(n),n ≠ 0。由[[alg_anelli|环]]的符号定理((−a)b = −(ab),a(−b) = −(ab),(−a)(−b) = ab),四种搭配下

x·y = ι(m·n)  或  x·y = −ι(m·n)。

由本节引理,m·n ≠ 0;ι 是单射且 ι(0) = 0,故 ι(m·n) ≠ 0。而 −ι(m·n) 也非零:若某 w 满足 −w = 0,两边取负得 w = −(−w) = −0 = 0(0 + 0 = 0 说明 −0 = 0)。两种情形,x·y ≠ 0。∎

**ℤ:含幺交换环、无零因子、1 ≠ 0——整环证书,签发完毕。** 随之而来的是环那节早已备案的推论:**整环中乘法消去律成立**——a ≠ 0 且 a·b = a·c ⟹ b = c。ℤ 没有乘法逆元,却能消去;「能消去」与「有逆元」并非一事,那句旧话在此处第一次有了亲手建成的见证。

## 回赠:ℕ 的乘法消去律

新世界向旧世界送出一份回礼。ℕ 自己的乘法消去律(k ≠ 0 且 k·m = k·n ⟹ m = n)我们从未证明;现在不必再回去做归纳——**借道 ℤ**:

k·m = k·n ⟹ ι(k)·ι(m) = ι(k)·ι(n)(ι 保乘法)⟹ ι(m) = ι(n)(ℤ 的整环消去律,ι(k) ≠ 0)⟹ m = n(ι 单射)。∎

在更大的世界里,定理更便宜——这正是「扩张」的另一重意义:不只为了解新方程,也为旧世界提供更省力的高地。(乐意的读者可以另走直路,对 ℕ 用三歧性与归纳直接证明,两条路殊途同归。)

## 一道激活练习

证明**乘负翻序**:设 x, y, z ∈ ℤ,x ≤ y 且 z ≤ 0,则

y·z ≤ x·z。

(提示:三步走。先由 z ≤ 0 得 0 ≤ −z,由 x ≤ y 得 0 ≤ y − x——两次「同加」;再用「非负乘非负」;最后把 (y−x)·(−z) 用环的符号定理展开,你会发现它恰是 x·z − y·z。途中需要一条小事实:−(u + v) = (−u) + (−v)——交换群里「和的逆是逆的和」,验证一行。)

## 参考解答

(建议先自己写完再看。)

**第一步,搬到原点两侧。** 由 z ≤ 0,两边同加 (−z):0 ≤ −z。由 x ≤ y,两边同加 (−x):0 ≤ y − x。

**第二步,非负相乘。** 由相容律,0 ≤ (y − x)·(−z)。

**第三步,展开。** 先备小事实:在交换群里 −(u+v) = (−u)+(−v)——验证 (u+v) + ((−u)+(−v)) = 0(交换结合重排),由逆元唯一性即得。于是

(y − x)·(−z) = −( (y − x)·z )  ⟨符号定理 a(−b) = −(ab)⟩
            = −( y·z + (−x)·z )  ⟨分配律⟩
            = −( y·z − x·z )  ⟨符号定理 (−x)z = −(xz)⟩
            = x·z − y·z。  ⟨和的逆是逆的和,再消双重负号⟩

**第四步,搬回去。** 0 ≤ x·z − y·z,两边同加 y·z:y·z ≤ x·z。∎

**顺手的收获:∀x ∈ ℤ,x² ≥ 0。** 三歧:若 0 ≤ x,相容律给 0 ≤ x·x;若 x ≤ 0,则 0 ≤ −x,而 x² = (−x)·(−x)(负负得正)≥ 0。那句在单射满射一节承诺已久的「平方非负」,在 ℤ 上首次成为定理——当初的字据写的是有序域,而这份证明只用了有序环的公理,届时逐字迁往 ℚ 与 ℝ,无需改动一字。

## 往前看一步

ℤ 已是有序的整环:能加减乘、能比大小、乘积为零必有零因。可它仍有一问无解:2·x = 1——「一半」无处安身,除法只对极少数搭配开放。药方你已用过一次,而且记得每个工序:到 ℤ × (ℤ∖{0}) 上去,让 (a, b) 代表「a/b 这个尚不存在的商」,以 a·d = c·b 宣布「商相同」,验等价、验良定义、商出新世界。传递性那一关,这次轮到**刚签发的整环消去律**站岗。开工:[[num_q|ℚ 的构造]]。
