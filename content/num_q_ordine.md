## 新问题:分母的符号

给 ℚ 移植序,直觉照旧:a/b ≤ c/d「应当」就是 ad ≤ cb——去分母,只用 ℤ 的乘法与序。但这次有个 ℤ 的构造里没有的问题:**分母可以为负**。同一个数 1/2 也写作 (−1)/(−2);若不加分辨地读分子,会把它误判成「负数」。去分母这一步,乘的是分母——而上一节练习刚证过,**乘以负数会翻转不等号**。结论:不整理代表,定义无从下笔。

处理并不难。

**引理(分母转正).** 每个有理数都有分母为正的代表:若 b < 0,则 (a, b) ~ (−a, −b),且 −b > 0。

**证明.** 等价性按定义验:a·(−b) 与 (−a)·b 由符号定理皆等于 −(ab)。又 b < 0 两边同加 −b 得 0 < −b。∎

从此约定:谈论序时,代表一律取分母为正者。

## 先备 ℤ 的两件小工具

去分母要反复「在不等式两边乘正数」,先把这两条工具备好。设 x, y, z ∈ ℤ。

**工具一(乘正保序).** x ≤ y 且 0 ≤ z ⟹ x·z ≤ y·z;且若 0 < u、0 < v,则 0 < u·v。

**证明.** 前半:由 x ≤ y 得 0 ≤ y − x;由相容律 0 ≤ (y−x)·z;按上一节练习里的展开,(y−x)·z = yz − xz;两边同加 xz 即得。后半:相容律先给 0 ≤ uv;若 uv = 0,整环无零因子迫使 u 或 v 为零,与严格正相抵;故 0 < uv。∎

**工具二(乘正消去).** x·z ≤ y·z 且 0 < z ⟹ x ≤ y。

**证明.** 由全序性,x ≤ y 或 y ≤ x。若是后者,工具一给 yz ≤ xz,与假设夹出 xz = yz(反对称),即 (x−y)·z = 0;z ≠ 0,整环消去,x = y——仍有 x ≤ y。两种情形俱得结论。∎

(两份证明只用了有序环的公理与整环性质,故它们在将来一切同类结构里照常生效。)

## 定义与良定义

**定义.** 设 x = a/b,y = c/d,代表已取 b > 0、d > 0。规定

x ≤ y :⟺ a·d ≤ c·b。

照例先过良定义检验——这是按代表元定义的**关系**,须验真假不随代表更换而变。

**定理(序的良定义).** 设 (a,b) ~ (a′,b′),(c,d) ~ (c′,d′),四个分母皆正,且 ad ≤ cb。则 a′d′ ≤ c′b′。

**证明.** 已知 ab′ = a′b,cd′ = c′d。b′ > 0、d′ > 0,由工具一后半,b′d′ > 0;在 ad ≤ cb 两边乘它(工具一前半):

(a·d)·(b′·d′) ≤ (c·b)·(b′·d′)。

左边重排:(ab′)(dd′) = (a′b)(dd′) = (a′d′)·(bd);右边重排:(cd′)(bb′) = (c′d)(bb′) = (c′b′)·(bd)。于是

(a′d′)·(bd) ≤ (c′b′)·(bd),

而 bd > 0,工具二消去,得 a′d′ ≤ c′b′。∎

与 ℤ 的同名定理对照:那里两边同加、再用加法消去;这里两边同乘正数、再用乘正消去。证明结构相同,使用的运算不同。

## 它是全序

四条性质,证明与 ℚ 等价关系那节平行,从简写过(设代表分母皆正):

**自反**:ab ≤ ab。**全序性**:ad 与 cb 是整数,ℤ 全序,必可比较。**反对称**:ad ≤ cb 与 cb ≤ ad 夹出 ad = cb——**恰是 (a,b) ~ (c,d)**,即同一个类;夹出的又是等价,商集中等价即相等,与 ℤ 一节相同。**传递**:设 ad ≤ cb、cf ≤ ed(b,d,f > 0)。第一式乘 f、第二式乘 b(工具一):

(af)·d = (ad)·f ≤ (cb)·f = (cf)·b ≤ (ed)·b = (eb)·d,

两端 (af)d ≤ (eb)d,d > 0,工具二消去:af ≤ eb。∎——结构仍与 ~ 的传递性证明平行,= 换作 ≤。

顺手补全嵌入的最后一项:**ι : ℤ → ℚ 保序**——m ≤ n ⟺ m·1 ≤ n·1 ⟺ m/1 ≤ n/1,两个方向同时成立。ℤ 连同序整体迁入,嵌入的全部性质至此核验完毕。

## 正性刻画与相容律

0 = 0/1,于是(取 b > 0)

0 ≤ a/b ⟺ 0·b ≤ a·1 ⟺ 0 ≤ a:

**分母转正之后,分数的正负由分子决定。** 由此两条相容律水到渠成:

**与加法**:x ≤ y ⟹ x + z ≤ y + z。取代表 a/b, c/d, e/f(分母皆正),按加法定义展开,要证 (af+eb)(df) ≤ (cf+ed)(bf);两边分配展开后,第二项相同,第一项是 ad·f² 与 cb·f² 之比较——f ≠ 0,由上一节的「平方非负」加无零因子得 f² > 0,工具一收尾。(上一节刚证的平方非负,在此已经派上用场。)

**与乘法**:0 ≤ x 且 0 ≤ y ⟹ 0 ≤ x·y。由正性刻画,分子 a ≥ 0、c ≥ 0,相容律给 ac ≥ 0,而 x·y = (ac)/(bd),分母 bd > 0,再由刻画即得。∎

## 有序域

现在给出正式定义:

**定义.** 一个**有序域**(ordered field;德语 angeordneter Körper)是一个域 K 连同其上的一个全序 ≤,满足两条相容公理:

- **(O1)** ∀x, y, z ∈ K:x ≤ y ⟹ x + z ≤ y + z;
- **(O2)** ∀x, y ∈ K:0 ≤ x ∧ 0 ≤ y ⟹ 0 ≤ x·y。

**定理.** ℚ(连同上面建立的序)是有序域。——域的部分上一节证毕,全序与两条相容本节逐一验讫。∎

随之,一批已证的定理对所有有序域生效:此前凡**只用有序环公理**证下的结论,在任何有序域中逐字成立——乘负翻序、平方非负,皆在此列。另有一条新的推论:

**推论.** 任何有序域中,0 < 1。

**证明.** 1 = 1·1 = 1²,由平方非负 0 ≤ 1;又 K1 说 1 ≠ 0。故 0 < 1。∎

这条推论的后果不小。两边各加 1,得 1 < 1+1,再加,1+1 < 1+1+1……于是 0 < 1 < 2 < 3 < ⋯ 永不重复——**任何有序域都是无穷的**。由此解决域那节遗留的问题:F₂ 里 1 + 1 = 0,若它能配上某个序成为有序域,将有 0 < 1 < 1+1 = 0,即 0 < 0,矛盾。**F₂ 上不存在任何使其成为有序域的全序**——「序」是实实在在的额外结构,并非每个域都具备;域那节那格 1 + 1 = 0 的反常,在此得到解释。

## 一道激活练习

设 K 是任意有序域(只许用公理与本节已证之事)。

(a) 证明:0 < x ⟹ 0 < x⁻¹。

(b) 证明:0 < x ≤ y ⟹ y⁻¹ ≤ x⁻¹。(取倒数,翻转不等号——但仅限同号的正数之间。)

(提示:(a) 先排除 x⁻¹ = 0,再反设 x⁻¹ < 0,用工具一(它在有序域照常生效)把这个反设与 x > 0 相乘,看与哪条刚证的推论矛盾。(b) 由 (a) 先取得 x⁻¹·y⁻¹ 的符号,然后在 x ≤ y 两边乘它。)

## 参考解答

(建议先自己写完再看。)

**(a)** 先排除零:若 x⁻¹ = 0,则 1 = x·x⁻¹ = x·0 = 0,与 K1 相抵,故 x⁻¹ ≠ 0。反设 x⁻¹ < 0,即 x⁻¹ ≤ 0 且不等:由乘正保序(乘以 x > 0),x⁻¹·x ≤ 0·x,即 1 ≤ 0;与推论 0 < 1 合看,反对称迫使 1 = 0,仍抵 K1。故只剩 0 < x⁻¹。∎

**(b)** 由 (a),x⁻¹ > 0 且 y⁻¹ > 0,工具一后半给 x⁻¹·y⁻¹ > 0。在 x ≤ y 两边乘 x⁻¹·y⁻¹(乘正保序):

x·(x⁻¹·y⁻¹) ≤ y·(x⁻¹·y⁻¹),

左边重排为 (x·x⁻¹)·y⁻¹ = y⁻¹,右边重排为 (y·y⁻¹)·x⁻¹ = x⁻¹。故 y⁻¹ ≤ x⁻¹。∎

这两条将来在分析中经常使用:谈「倒数的极限」「分式的估计」,引用的正是它们。

**推论(稠密性 density).** 设 x < y,则存在 m ∈ ℚ 使 x < m < y。证明:令 2 := 1 + 1(由 0<1 两边加 1 得 1 < 2,传递 0 < 2),由 (a),2⁻¹ > 0。取中点 m := (x + y)·2⁻¹。则

m − x = (x + y)·2⁻¹ − x = (x + y − 2x)·2⁻¹ = (y − x)·2⁻¹ > 0,

(末步:y−x > 0 与 2⁻¹ > 0 严格相乘,工具一后半)即 x < m;对称地 y − m = (y−x)·2⁻¹ > 0,即 m < y。**任意两个有理数之间,必有第三个**——再往两个缝里各取一个,无穷无尽:

## 往前看一步

总结 ℚ 的性质:域,全序,序与运算相容,处处稠密。仅从这些性质看,这条数轴似乎不该再有空缺。正因如此,下一节的定理才出人意料:**没有任何有理数,平方等于 2。** 一个边长为 1 的正方形,它的对角线长度在 ℚ 的数轴上没有对应的点——处处稠密,却仍有缺口。这个事实对古希腊数学曾是一次冲击;对我们,它是 ℚ 的缺口的精确定位,也是通往 ℝ 的第一步:[[num_radice2|√2 不是有理数]]。
