## 「视为相同」

数学里有个挥之不去的需求：把**不完全相同**的东西**当作相同**来处理。分数 1/2 和 2/4 写法不同，却该算同一个数；两个三角形位置不同，只要能叠合，就该算「同一个形状」；统计奇偶时，3 和 17 没有分别。每次我们都在说同一句话——「在我此刻关心的层面上，它们一样」。

可「视为相同」要是随手乱用，立刻出乱子：要是「视为相同」连自己跟自己都不算相同，或者 a 同 b、b 同 c 却 a 不同 c，那「按相同归堆」这件事根本做不成。要让归堆站得住，「视为相同」必须守纪律——而需要的纪律，恰恰就是上一节备好的那三条。这绝非巧合：三条品性本来就是从「相等」身上提炼出来的，等价关系就是「**像相等一样行事**的关系」。

## 定义与例子

设 ~ 是集合 A 上的一个 [[fond_relazioni|关系]]。若 ~ 同时满足

- 自反：∀x ∈ A, x ~ x；
- 对称：∀x,y ∈ A ( x ~ y ⟹ y ~ x )；
- 传递：∀x,y,z ∈ A ( (x ~ y ∧ y ~ z) ⟹ x ~ z )，

就称 ~ 是 A 上的一个**等价关系**(equivalence relation)。习惯用波浪号 ~ 作记号，读「等价于」。

例子（最后两个反例照例是命脉）：

- **相等** = 本身：三条全占，是「最细」的等价关系——谁也不跟别人等价，每个元素只跟自己一堆。
- **全关系** A×A：人人等价于人人，三条也全占，是「最粗」的——全体挤成一堆。等价关系全体就活在这两个极端之间。
- **同奇偶**：自然数上 x ~ y ⟺ x、y 同为偶数或同为奇数。三条都成立（同自己同奇偶；同奇偶不分先后；接力两次奇偶不变）。自然数与奇偶我们暂且按你熟悉的方式用，它们的严格重建留待 [[num_n|自然数]]。
- **「同一年出生」**（某群人上）：三条全占——日常语言里一切「同……」式的说法，几乎都是等价关系，这正是「视为相同」的语感来源。
- **反例 ≤**（ℝ 上）：自反、传递，但**不对称**（2≤3 而 3≰2）。它不是等价关系——它是另一个重要物种（序关系）的代表，那条线以后再走。
- **反例「是朋友」**：对称，未必传递（朋友的朋友不一定是朋友）。守不住接力，归堆就会归出含混的边界——这正是传递性不可或缺的原因。

## 等价类

有了守纪律的「视为相同」，归堆就可以动手了。对 a ∈ A，把所有与 a 等价的元素收进一个集合：

[a] := { x ∈ A : x ~ a }，

称为 a 的**等价类**(equivalence class)，a 叫这个类的一个**代表元**(representative)。

同奇偶的例子里只有两个类：[0] = {0, 2, 4, …}（全体偶数）和 [1] = {1, 3, 5, …}（全体奇数）；并且 [0] = [2] = [4] = ⋯——**同一个类可以由它的任何成员代表**。这句话不该轻信，得证：

**引理.** 设 ~ 是 A 上的等价关系，a, b ∈ A。则
(i) a ∈ [a]；
(ii) a ~ b ⟺ [a] = [b]。

**证明.** (i) 即 a ~ a，自反性原话。（别小看这条：它保证**每个类都不空**，且每个元素都至少落在一个类里——落在自己的类里。）

(ii) 「⟸」：设 [a] = [b]。由 (i)，a ∈ [a] = [b]，按定义即 a ~ b。

「⟹」：设 a ~ b，证两个集合相等。按上一节的双向夹，分证两个包含。先证 [a] ⊆ [b]：任取 x ∈ [a]，即 x ~ a；又 a ~ b，由**传递**得 x ~ b，即 x ∈ [b]。再证 [b] ⊆ [a]：任取 x ∈ [b]，即 x ~ b；由 a ~ b 与**对称**得 b ~ a，再由**传递**得 x ~ a，即 x ∈ [a]。两个方向都成立，由外延原则 [a] = [b]。∎

注意证明里三条性质各司其职、一条不多一条不少——这是「为什么定义恰好要这三条」的最好答案。

## 类把集合切成划分

把一个集合「切干净」这件事，先给个正式名字。A 的一个**划分**(partition)是一族非空子集，满足两条：**两两不相交**（任两块要么相等要么交为空），且**并起来盖满 A**（每个元素都落在某块里）。直观就是切蛋糕：不留缝、不重叠。

**定理.** A 上任一等价关系 ~ 的全体等价类，构成 A 的一个划分。

**证明.** 三件事。**非空**：引理 (i)，每个类含自己的代表元。**盖满**：任给 a ∈ A，引理 (i) 说 a ∈ [a]——每个元素都被某个类接住。**两两不相交**：要证任两类要么相等、要么交为空——等价地说，**只要有一个公共成员，两类就整个重合**。这一条留作下面的激活练习（引理 (ii) 已把刀磨好）。∎

反过来也对：给 A 任意一个划分，规定「x ~ y ⟺ x 与 y 落在同一块里」，立刻得到一个等价关系，其等价类恰是原来的那些块。所以**等价关系与划分是同一件事的两种说法**：一个用「关系」的语言，一个用「切块」的语言，信息完全相同。

## 商集：把「视为相同」升格为「真的相同」

最后一步是全节的灵魂。把所有等价类**本身**收成一个新集合：

A / ~ := { [a] : a ∈ A }，

读「A 模 ~」，叫**商集**(quotient set)。

体会一下发生了什么：在 A 里，3 和 17 是两个不同的元素，只是「被视为」相同（同奇偶）；可在商集 ℕ/~ 里，[3] 和 [17] **就是同一个元素**——引理 (ii) 白纸黑字担保了 [3] = [17]。差异没有被假装看不见，而是被**整个打包进类里、再把类当成新的个体**。「视为相同」就此升格为「真的相同」。商集 ℕ/~ 恰有两个元素：{[0], [1]}——它就是「奇偶性」这个概念本身的集合论肉身。

这台机器的威力在于**无中生有地制造新对象**。预告一处你很快会亲眼看到的工地：整数。我们手里将只有自然数，而 3 − 5 在自然数里无家可归；出路是拿有序对 (a, b) 充当「a − b 的欠条」，宣布 (3, 5) 与 (4, 6) 这些「差相同」的欠条互相等价，然后——商掉。等价类 [(3,5)] 这个**集合本身**，就是将来那个叫「−2」的对象。ℚ 的造法如出一辙（(1,2) 与 (2,4) 等价……）。每次的套路都一样：先在旧材料上铺一张等价关系，再取商集，新数就诞生了。上一节说的「造出来→验证性质→封装遗忘」，配上这台机器，就是数系构造的全部秘密。

## 一道激活练习

补上定理缺的那块拼图。设 ~ 是 A 上的等价关系，a, b ∈ A。证明：

若 [a] ∩ [b] ≠ ∅，则 [a] = [b]。

（提示：交不空意味着能**取到**一个公共成员 c——这是 ∃ 的用法；然后把 c ∈ [a]、c ∈ [b] 按定义翻译成 ~ 的语句，用对称、传递把 a ~ b 凑出来，最后让引理 (ii) 收口。）

## 参考解答

（建议先自己写完再看。）思路：交不空 → 取见证 c → 翻译成关系语句 → 对称＋传递凑出 a ~ b → 引理 (ii) 一锤定音。

设 [a] ∩ [b] ≠ ∅。按非空的含义，存在 c ∈ [a] ∩ [b]，即 c ∈ [a] 且 c ∈ [b]。按等价类的定义翻译：

```
c ∈ [a]  即  c ~ a
c ∈ [b]  即  c ~ b
```

由 c ~ a 与**对称性**得 a ~ c；又 c ~ b，由**传递性**得 a ~ b。最后由引理 (ii)（a ~ b ⟺ [a] = [b]），得 [a] = [b]。∎

一句话回味：两个类只要碰了一下，就被对称＋传递「焊」成一整块——这正是「不留缝、不重叠」里「不重叠」的来源；而「不留缝」来自自反性（人人都被自己的类接住）。三条纪律，各管划分的一半江山。

## 往前看一步

关系这条线上，我们已经收编了「守纪律的视为相同」。但还有另一种关系，纪律的方向完全不同：不求对称，反倒要求**每个 a 恰好指向一个 b**——不许多指、不许不指。这种「精确制导」的对应，就是整个数学里出镜率最高的角色：[[fond_funzioni|函数]]。
