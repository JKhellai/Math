## 两种运算如何同居

群刻画一种运算的良好品行。可算术从来不是独奏:ℕ 上加法与乘法并肩而立,且由分配律 k·(m+n) = k·m + k·n 锁在一起——那条我们在[[num_aritmetica|自然数的算术]]里亲手证过的定律,正是两种运算之间唯一的、也是全部的纽带:没有它,加与乘不过是同一集合上互不相识的两台机器,「乘法是重复的加法」这层联系将无从谈起。把「一个加法群 + 一个结合的乘法 + 分配律」提炼成公理,就得到代数的第二座主要建筑。

## 定义

**定义.** 一个**环**(ring)是一个三元组 (R, +, ·),其中 R 是集合,+ 与 · 是 R 上的两个二元运算,满足:

- **(R1)** (R, +) 是交换群。其单位元记 **0**(称零元),a 的加法逆记 **−a**。
- **(R2)** · 满足结合律:∀a, b, c ∈ R, (a·b)·c = a·(b·c)。
- **(R3) 分配律(distributivity)**,左右两条:∀a, b, c ∈ R, a·(b+c) = a·b + a·c,且 (a+b)·c = a·c + b·c。

若乘法另有单位元 **1**(∀a, 1·a = a = a·1),称**含幺环**(ring with identity);若乘法交换,称**交换环**(commutative ring)。(各家著作约定不一:有的把含幺写进环的定义。我们按上述办;翻阅他书时留意核对——这类约定差异在数学里寻常,警觉即可。)

请读出定义里的**不对称**:对加法,我们索要一个完整的交换群;对乘法,只要结合律——不要求逆元,甚至不要求单位元。这不是吝啬,是忠实于样本:将来的 ℤ 里,乘法的逆几乎无人拥有(2 的倒数不是整数),公理若强行索要,就把最重要的样本拒之门外了。公理化的分寸,正在于向真实的样本看齐。

分配律为何要左右两条?因为乘法未必交换:若交换,一条可推出另一条;一般情形二者独立,缺一不可。

记号照旧从简:a·b 写 ab;乘法优先于加法(ab + c 读作 (a·b) + c);并且——请留意这一刻——定义

a − b := a + (−b)。

**减法在任何环中天然存在。** ℕ 所缺的那件东西,在环里由 R1 直接担保:这并非额外规定,而是「加法成交换群」的逻辑后果——方程 x + a = b 在环中恒有唯一解 x = b − a([[alg_gruppi|群]]那节练习的定理,换上加法记号)。当然,眼下我们手里还没有一个装着全体整数的环;把 ℤ 造出来并逐条核验 R1–R3,是不远处的工程。此处先把图纸画好。

## 例子:把逻辑抬升成环

含两种运算的严格样本,手头的材料已足够搭出一个,而且来路漂亮。取任意集合 X,在幂集 P(X) 上设两种运算:乘法取交 ∩;加法取**对称差**(symmetric difference)

A Δ B := (A ∖ B) ∪ (B ∖ A)

——「恰属其一」的元素全体。还记得命题那节辨析过的[[fond_proposizioni|排斥或]]么:x ∈ A Δ B 当且仅当「x ∈ A 与 x ∈ B 恰有一真」。于是这一对运算正是逻辑联结词的又一次抬升,与当年「∪∩∖ 即 ∨∧¬」一脉相承:

| 集合层 | 逻辑层 |
|---|---|
| A Δ B | 排斥或(恰一为真) |
| A ∩ B | ∧ |
| ∅ | 恒假 |
| X | 恒真 |

**断言.** (P(X), Δ, ∩) 是含幺交换环:零元为 ∅,幺元为 X,且每个 A 的加法逆是 **A 自己**——A Δ A = ∅,因为「恰属其一」对同一个集合无人能满足。

核验的路数全是旧法:把成员资格翻译到逻辑层,化为真值表可验的等价,再用[[fond_insiemi|外延原则]]收口。示范最短的一条——A Δ ∅ = A:x ∈ A Δ ∅ ⟺ 「x∈A 与 x∈∅ 恰有一真」⟺ x ∈ A(因 x ∈ ∅ 恒假)。其余各条(Δ 的结合律与交换律、∩ 对 Δ 的分配律)同一路数,只是真值表更长;Δ 的结合律是其中最考耐心的一条,愿意磨一遍的读者会得到一次扎实的训练。

这个环的性质与你将来熟识的数环很不同:人人是自己的加法逆(a = −a),还藏着文末要谈的另一桩怪事——它提醒我们,公理圈定的范围比「数」广得多。

## 第一批定理:零元与负号的规矩

与群那节一样,以下只用公理推演、不看元素的脸,故对一切环生效。群的两条定理将反复出场:消去律,与逆元的唯一性。

**定理 1.** ∀a ∈ R, 0·a = 0 = a·0。

**证明.** 由零元的资格,0 + 0 = 0。两边右乘 a,左边用分配律展开:

0·a + 0·a = (0 + 0)·a = 0·a = 0·a + 0。

视作交换群 (R, +) 中的等式 0·a + 0·a = 0·a + 0,左消去 0·a,得 0·a = 0。另一半同理(左乘并用另一条分配律——两条分配律在此各使用一次)。∎

读出它的意味:**乘法公理对零元只字未提,分配律却强迫乘法尊重加法的零**——「乘以零得零」不是规定,而是分配律带来的必然。

**定理 2.** ∀a, b ∈ R, (−a)·b = −(a·b) = a·(−b)。

**证明.** 验证 (−a)·b 配得上「a·b 的加法逆」这一身份:

a·b + (−a)·b = (a + (−a))·b = 0·b = 0,

三步依次是:分配律、逆元、定理 1。而加法逆唯一(群那节的定理 2),故 (−a)·b = −(a·b)。另一半同理。∎

这条定理的日常面孔是「负号可以从乘积里提出」。它的著名推论——那句小学口诀——留给你亲手确立。

## 一道激活练习

设 R 是环,a, b ∈ R。证明

(−a)·(−b) = a·b。

「负负得正」从口诀升格为定理的一刻。(提示:两步。先用定理 2 把 (−a)·(−b) 化成 −(某物);随后只剩一问——−(−x) 是什么?在群里,「x 的逆」的逆是谁,逆元的唯一性会替你回答。)

## 参考解答

(建议先自己写完再看。)

第一步,提出左因子的负号(定理 2 之「(−a)·c = −(a·c)」,取 c = −b),再提出右因子的(定理 2 另一半):

(−a)·(−b) = −( a·(−b) ) = −( −(a·b) )。

于是只剩一问:−(−(a·b)) 是否等于 a·b?

设 x ∈ R。等式 x + (−x) = 0 有两种读法:它既说 −x 是 x 的加法逆,也说 **x 是 −x 的加法逆**。而群中逆元唯一——−x 的逆只有一个,记作 −(−x);既然 x 具此资格,便有 −(−x) = x。取 x = a·b,得 −(−(a·b)) = a·b。

合起来:(−a)·(−b) = a·b。∎

回味:这句口诀的根,一半在分配律(定理 2 的来源),一半在逆元唯一性。在公理体系中,学校里背诵的规则没有被「规定」的位置——它要么被证明,要么不成立;此处,它被证明了。

## 零因子

环的公理对乘法要求宽松,这有相应的后果。数的世界里你习惯了「两数之积为零,必有一者为零」;环并不担保这一点。就在上面的幂集环里:取 X = {1, 2},则 {1} ∩ {2} = ∅——两个**非零**元素(谁也不是 ∅),乘积却是零元。这样的元素称**零因子**(zero divisor)。

没有零因子、含幺、交换、且 1 ≠ 0 的环,称**整环**(integral domain)——「整」字预告了它的样板:将来的 ℤ 正是整环,而「ℤ 无零因子」届时会是一条需要证明的定理。整环里,乘法的消去律失而复得:a ≠ 0 且 ab = ac,则 a·(b − c) = ab − ac = 0,无零因子便强迫 b − c = 0,即 b = c——尽管乘法逆元依旧缺席。可见「能消去」与「有逆元」并非一事:前者只要求没有零因子,后者是更强的条件。

## 往前看一步

环里加法十全十美,乘法却仍留两处余地:未必有幺元,非零元未必可逆。把这两处余地也一并取消——要求 1 存在、且每个非零元素都有乘法逆——乘法便在 R ∖ {0} 上自成一个交换群,加乘两边终于平起平坐。ℚ 与 ℝ 都将以这种结构的身份登场:[[alg_campi|域]]。
