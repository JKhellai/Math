## 为什么需要集合

上一节的量词总要「取遍某个范围」——∀x、∃x 里那个 x 究竟在哪儿取？我们一直含糊地说「论域」「一堆对象」。现在把这个范围本身钉死：它就是**集合**。集合是数学里**最通用的容器**——往后每一个对象（数、函数、空间里的点）都将被装进某个集合里来谈。

## 集合与属于

**集合**(set) 就是把若干对象看作一个整体；这些对象叫它的**元素**(element)。最根本的关系是**属于**(membership)，记 x ∈ A，读作「x 是 A 的元素」；不属于记 x ∉ A。

属于关系 ∈ 我们**不再往下定义**——它和逻辑一样，是这一层的原始概念，我们约定直接使用。（这是个选择：整个集合论就架在 ∈ 这一个原始关系之上。）

怎么写出一个具体的集合？两种办法：

- **列举**：把元素列出来，如 A = {0, 1, 2}。花括号里不计次序、不计重复——{0,1,2}、{2,1,0}、{0,0,1,2} 是同一个集合（凭什么算「同一个」，下面外延原则会讲死）。
- **描述**：从一个**已有的**集合 A 里，挑出满足某谓词 P(x) 的元素，记 {x ∈ A : P(x)}。这里直接用上了上一节的 [[fond_quantificatori|谓词]]——例如 {x ∈ ℝ : x > 3}。

> 留意「从已有集合里挑」这个限制：不能无中生有地写 {x : P(x)}（对「一切满足 P 的 x」一网打尽）。这种不加约束的写法会闹出矛盾（罗素悖论），严格的回避之道留待后面专门处理；眼下我们一律**从一个已经到手的集合里切**，就安全了。

## 外延原则：集合由成员决定

两个集合什么时候**相等**？回答只有一条——**外延原则**(extensionality)：两个集合相等，当且仅当它们含有完全相同的元素。用我们已有的逻辑语言写出来，干净利落：

A = B  ⟺  ∀x ( x ∈ A ⟺ x ∈ B )。

这条把「相等」彻底交给了「成员」：集合**不在乎**元素怎么排、写了几遍、用什么方式描述出来——只认「谁在里面」。这就是为什么 {0,1,2} 和 {2,1,0} 是同一个集合。

## 子集

若 A 的每个元素都还在 B 里，就说 A 是 B 的**子集**(subset)，记 A ⊆ B。又是用 ∀ 和蕴含直接拼出来：

A ⊆ B  ⟺  ∀x ( x ∈ A ⟹ x ∈ B )。

把它和外延原则并到一处，得到一条天天要用的「双向夹」：

A = B  ⟺  ( A ⊆ B  ∧  B ⊆ A )。

（因为前面讲联结词时说过 p ⟺ q 恰是 (p⟹q)∧(q⟹p)，把它套到 x∈A ⟺ x∈B 上即得。）所以证集合相等的标准套路就是：**证 A⊆B、再证 B⊆A**，两个方向各跑一遍——你以后会无数次这么做。

## 基本运算就是联结词

这是本节最漂亮的一点：集合的交、并、差，**根本不是新东西**——它们就是讲联结词那节的 [[fond_proposizioni|联结词]] ∧、∨、¬，被「抬」到属于关系 ∈ 上：

- **并**(union)：x ∈ A ∪ B  ⟺  x ∈ A  ∨  x ∈ B。
- **交**(intersection)：x ∈ A ∩ B  ⟺  x ∈ A  ∧  x ∈ B。
- **差**(difference)：x ∈ A ∖ B  ⟺  x ∈ A  ∧  ¬(x ∈ B)。

一旦「成员资格」被翻译成逻辑命题，集合的恒等式就**全部塌缩成上面那些逻辑等价**。举个例子，集合版的德摩根律 A∖(B∪C) = (A∖B)∩(A∖C)，证起来无非是把 ∈ 翻成逻辑、套一次逻辑的德摩根、再用外延原则收口。这就是套路：**集合代数 = 逻辑代数，根都在 ∈ 和那张真值表里**。

## 空集

有一个集合一个元素都没有，叫**空集**(empty set)，记 ∅。它有个初看古怪的性质：**∅ 是任何集合的子集**，即 ∅ ⊆ A 对一切 A 成立。

为什么？按定义，∅ ⊆ A 就是 ∀x ( x ∈ ∅ ⟹ x ∈ A )。可 x ∈ ∅ 永远为假，于是这个蕴含的前件恒假——它**空洞地为真**（正是讲蕴含时「前件为假，蕴含自动成立」那一条）。前件从不成立，承诺就从不被推翻，所以 ∅ ⊆ A 自动成立。逻辑那一层埋下的种子，在这里结了第一个果。

## 一道激活练习

用**外延原则**证明交运算可交换：

A ∩ B = B ∩ A。

（思路提示：要证两集合相等，按外延原则只需证 ∀x ( x∈A∩B ⟺ x∈B∩A )；把两边的 ∈ 都翻成逻辑，你会发现它归结为前面某条关于 ∧ 的等价。）

## 参考解答

（建议先自己写完再看。）思路：集合相等 → 外延原则 → 把成员资格翻成逻辑 → 套一条逻辑等价 → 收口。

由外延原则，要证 A∩B = B∩A，只需对任意 x 证 x∈A∩B ⟺ x∈B∩A。逐步：

```
  x ∈ A∩B
⟺ x∈A ∧ x∈B       （交的定义）
⟺ x∈B ∧ x∈A       （∧ 的交换律——用真值表一眼可验：p∧q 与 q∧p 两列相同）
⟺ x ∈ B∩A         （交的定义，反着用）
```

对一切 x 成立，故由外延原则 A∩B = B∩A。∎

一句话思路：**集合的「交换律」其实是「且」的交换律换了身衣服**——这正是「集合代数 = 逻辑代数」的一个实例。同样地，集合的德摩根律、分配律，根也都在前面那些逻辑等价里。

## 往前看一步

现在我们有了容器（集合），也能判断相等、包含，做交并差了。但眼下集合里的元素还各自孤立；数学真正关心的，是元素之间的**对应**——把 A 里的东西按某种规则关联到 B 里的东西。要精确地谈「对应」，得先有一个能装下「一对有次序的对象」的工具，再从中提炼出**关系**与函数。这就引向接下来的 [[fond_relazioni|关系]]。
