## 同一副面孔,第三次出现

回想两处看似无关的风景。其一,[[fond_composizione|复合与逆]]那节:函数的复合满足结合律,恒等函数是单位,双射人人可逆——当时特意提请留意这套组合。其二,上一节的 (ℕ, +):结合律有了,单位元 0 有了,唯独逆元缺席——x + 3 = 1 无解,正是这个缺口让 ℕ 显得残缺。

两处风景,元素的身份毫不相干——一处是函数,一处是数——可**运算遵守的规矩**几乎逐条重合。十九世纪的代数学家们由此走出了关键的一步:不再问「元素是什么」,只问「运算守什么规矩」。把规矩本身写成公理、研究一切守这套规矩的世界——这就是代数(algebra)的开端,而第一套规矩,叫群。

## 定义

先把「运算」说严格。集合 G 上的一个**二元运算**(binary operation),就是一个函数 ∗ : G × G → G,习惯把 ∗((a,b)) 写作 a ∗ b。注意「函数到 G」这五个字已经悄悄办妥了一件事:任意两个元素运算的结果**仍落在 G 内**(封闭性)——它不必单列为公理,是「运算是函数」的直接后果。这是函数语言的又一次效力。

**定义.** 一个**群**(group)是一个偶对 (G, ∗),其中 G 是非空集合,∗ 是 G 上的二元运算,满足三条公理:

- **(G1) 结合律(associativity)**:∀a, b, c ∈ G, (a ∗ b) ∗ c = a ∗ (b ∗ c)。
- **(G2) 单位元(identity element)**:∃e ∈ G, ∀a ∈ G, e ∗ a = a = a ∗ e。
- **(G3) 逆元(inverse element)**:∀a ∈ G, ∃b ∈ G, a ∗ b = e = b ∗ a。

(G3 里的 e 指 G2 所给的那一个;严格地讲,G2 与 G3 应连写成一个嵌套的量词语句「∃e (… ∧ ∀a ∃b …)」——稍后证明了 e 的唯一性,这层小心就可以放下了。)

三条公理无一是新面孔:结合律与单位元在复合那节逐字见过,逆元正是 ℕ 所缺的一条。群,就是「**可撤销的运算**」的公理化:任何一步操作,都存在恰好抵销它的另一步。

若再多守一条**交换律** ∀a,b, a ∗ b = b ∗ a,称 (G, ∗) 为**交换群**,或依传统称**阿贝尔群**(abelian group)。注意:交换律**不在**群的定义之内——这不是疏忽,是刻意。下面立刻会看到一个堂堂正正、却不交换的群。

记号约定:抽象讨论时常省去 ∗,把 a ∗ b 写作 ab,单位元写 e,a 的逆写 a⁻¹(待证唯一后,这个记号才算名正言顺);若运算本身是某种「加法」,则改写 +、0、−a。记号随场合换装,公理始终是同三条。

## 例子

**主例:一个集合的全体双射。** 取任意非空集合 X,令 S(X) 为 X 到自身的全体双射所成的集合,运算取复合 ∘。逐条验收,而且**每条都是已经建立的定理**:复合的衔接合法且双射的复合仍是双射(故 ∘ 确是 S(X) 上的二元运算);结合律——复合那节的定理;单位元——id_X;逆元——「可逆 ⟺ 双射」。一条都不必新证,(S(X), ∘) 是群,称 X 上的**对称群**(symmetric group)。请体会这一刻:我们的第一个群,完全用地基里的材料搭成,没有借用任何尚未建立之物。

最小的非平凡样本:X = {1, 2}。S(X) 恰有两个元素——恒等 id 与对换 τ(把 1、2 互换)。全部运算结果列成一张表(行 ∗ 列):

| ∘ | id | τ |
|---|----|---|
| **id** | id | τ |
| **τ** | τ | id |

读出几件事:τ ∘ τ = id,即 τ 的逆是它自己——「撤销一次对换」就是再换一次;整张表关于对角线对称,这个小群是交换的。

**平凡群**:单元素集 {e},运算只能是 e ∗ e = e。三条公理空洞而严格地成立——最小的群。

**借用的预览**(这些数系尚待构造,公理逐条核验留到各自建成之日):(ℤ, +) 是交换群,0 为单位、−a 为逆;(ℚ∖{0}, ·) 是交换群,1 为单位、倒数为逆——注意必须剔除 0,它没有乘法逆。

**反例:(ℕ, +) 不是群。** G1、G2 成立而 G3 不成立:除 0 外无人有逆——若 n + m = 0 而 n ≠ 0,写 n = S(k),则 S(k) + m = S(k + m) = 0,与 Peano 公理「0 不是后继」相抵。顺带一说,「结合律 + 单位元、唯缺逆元」的结构也有自己的名字,**幺半群**(monoid)——ℕ 在 + 下正是一个交换幺半群。它并非残次品,只是另一物种;但「解方程」这件事,确实要等逆元到位才谈得上。

**反例:三个元素以上,对称群不交换。** 取 X = {1, 2, 3},f 对换 1、2 而固定 3,g 对换 2、3 而固定 1。算两个复合在 1 处的值:(g ∘ f)(1) = g(2) = 3,而 (f ∘ g)(1) = f(1) = 2。在 1 处取值已不同,故 g ∘ f ≠ f ∘ g。先做哪一步,结果不同——交换律是额外的假设,不在群的公理之内。

## 第一批定理:在公理上行走

现在展示这一步抽象的效力。以下证明**只准使用三条公理**,不许看元素的脸——因此它们一经写下,就同时在对称群、在 ℤ、在将来你遇到的每一个群里生效。

**定理 1(单位元唯一).** 群中满足 G2 的元素只有一个。

**证明.** 设 e、e′ 都满足 G2。把 e′ 当单位元用:e = e ∗ e′;把 e 当单位元用:e ∗ e′ = e′。故 e = e′。∎ (两行,但请看清每行各动用了谁的资格。)

**定理 2(逆元唯一).** 对任意 a ∈ G,满足 G3 的 b 只有一个。

**证明.** 设 b、b′ 都是 a 的逆。则

b = b ∗ e = b ∗ (a ∗ b′) = (b ∗ a) ∗ b′ = e ∗ b′ = b′。∎

请把这串等式与复合那节「逆函数唯一」的证明并排放——**一字不差**。当时我们以为是在证一条关于函数的引理;今天才看清,那其实是一条群的定理,我们早已在群里工作,只是尚无此名。这正是公理化方法的全部力量:一次证明,处处生效。从此 a⁻¹ 这个记号名正言顺。

**定理 3(消去律 cancellation laws).** 在群中,a ∗ b = a ∗ c ⟹ b = c;同样 b ∗ a = c ∗ a ⟹ b = c。

**证明.** 设 a ∗ b = a ∗ c。两边左乘 a⁻¹:a⁻¹ ∗ (a ∗ b) = a⁻¹ ∗ (a ∗ c);结合律挪括号:(a⁻¹ ∗ a) ∗ b = (a⁻¹ ∗ a) ∗ c;即 e ∗ b = e ∗ c,即 b = c。右侧同理(右乘 a⁻¹)。∎

消去律值得一句旁白:在 ℕ 里「两边消去同一个加数」当然也对,但那需要对 ℕ 的具体构造做一番归纳;在群里它是三行公理推演。具体构造里费力的事实,在公理层面常常一步可达——选好公理,常能大幅缩短证明。

## 一道激活练习

设 (G, ∗) 是群,a, b ∈ G。证明方程

a ∗ x = b

在 G 中**恰好有一个**解,并把解写出来。(提示:「恰好一个」照旧拆作存在与唯一两半。存在性——动手构造,想想该往 b 的左边搁什么;唯一性——定理 3 正候着。也请顺带想一想:这道题在 (ℕ, +) 里为何会失败,失败在哪一半?)

## 参考解答

(建议先自己写完再看。)

**存在性.** 取候选 x₀ := a⁻¹ ∗ b,验证:

a ∗ (a⁻¹ ∗ b) = (a ∗ a⁻¹) ∗ b = e ∗ b = b。

三步分别用了结合律、G3、G2——候选合格。

**唯一性.** 设 x、x′ 都满足 a ∗ x = b = a ∗ x′。由定理 3(左消去),x = x′。∎

故解恰为 x = a⁻¹ ∗ b。在 (ℕ, +) 中,方程 3 + x = 1 失败在**存在性**那一半:构造解需要 a 的逆,而 ℕ 里 3 没有逆——并非我们不够聪明,是材料里确实没有这味原料。如何在 ℕ 的基础上**严格地补足**这味原料而不只是口头宣布,正是数系构造的下一项工程。

## 往前看一步

群管一种运算。可你最熟悉的那些数的世界里,从来是**两种**运算并肩而立——加法与乘法,而且二者绝非各自为政:分配律 k·(m+n) = k·m + k·n(上一节刚刚证过的那条)把它们锁在一起。一个集合,两种运算,以及一条联结二者的定律——这样的结构该如何公理化,各公理又各司何职,这是 [[alg_anelli|环]] 要回答的问题。
