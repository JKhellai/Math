## 不数数,怎么比多少

一个上古的难题:牧人不识数,怎么知道羊有没有丢?办法是**配对**——出栏时每只羊扔一颗石子进罐,归栏时每只羊取一颗出来;罐子空了,羊就齐了。他自始至终不知道羊「有几只」,却完全有把握羊和石子「一样多」。

这给了我们一个深刻的启示:**「一样多」比「有几个」更原始**——比较不需要先数数,配对就够了。而「完美配对」恰恰是我们刚刚锻造完的工具:[[fond_iniettive_suriettive|双射]]。于是一个古老的直觉,今天升格为定义。

## 等势

两个集合 A、B 称为**等势**(equipotent),记 A ≈ B,若存在一个双射 f : A → B。

它的行为像极了[[fond_equivalenza|等价关系]],三条全占——而且每一条都恰好由上一节的一件存货担保:

- **自反** A ≈ A:恒等函数 id_A 是双射。
- **对称** A ≈ B ⟹ B ≈ A:双射 f 可逆,而 f⁻¹ 自己也是双射——因为 f⁻¹ 有逆(就是 f),由[[fond_composizione|可逆 ⟺ 双射]]即得。
- **传递** A ≈ B ∧ B ≈ C ⟹ A ≈ C:双射的复合是双射(上节引理 1+2)。

> 一句诚实话:严格说,等价关系是「某个集合**上**的关系」,可「所有集合的集合」不是合法对象——还记得集合那节按下的罗素悖论么,这正是它的回响。所以我们说等势「像」等价关系:三条性质逐字成立、放心使用,只是别问「它是哪个集合上的关系」。

## 有限、无穷、可数

有了「一样多」,就能给「有 n 个」正身:A 是**有限的**(finite),若 A = ∅ 或 A ≈ {1, 2, …, n} 对某个自然数 n 成立;否则称**无穷的**(infinite)。数数,原来就是和样板集 {1,…,n} 配对。(自然数照例先按熟悉的方式借用——它的严格出生证明在不远处等着我们。)

无穷集合里地位最特殊的标尺是 ℕ 自己。与 ℕ 等势的集合称**可数无穷**(countably infinite);有限或可数无穷统称**可数**(countable)。「与 ℕ 等势」说白了就是**能排成一列不重不漏的清单** a₀, a₁, a₂, …——双射 ℕ → A 就是这张清单,n ↦ 第 n 号展品。

## 第一记重锤:部分可以和整体一样多

现在让无穷亮出獠牙。看偶数集 E = {0, 2, 4, …}。它是 ℕ 的**真子集**——奇数全被剔除,「少了一半」。可是:

**断言.** ℕ ≈ E,由 f : ℕ → E, n ↦ 2n 实现。

**证明.** 单射:设 2n = 2m,除以 2 得 n = m。满射:任给 e ∈ E,按 E 的定义 e = 2k 对某个 k ∈ ℕ,于是 f(k) = e。两条都过,f 是双射。∎

证明短得让人不安,结论却推翻了一条两千年的「公理」——欧几里得说「整体大于部分」。在有限世界这没错;在无穷世界,**整体可以和真部分一样多**。伽利略在十七世纪就撞见了这件事(平方数与全体自然数配对),被吓得退了回去;Cantor 的过人之处在于不退:既然定义推出如此,那就接受它——这不是悖论,是无穷的**本性**。(事实上这条性质刻画了无穷:一个集合是无穷的,当且仅当它与某个真子集等势——Dedekind 干脆拿它当无穷的定义。)

更夸张的:ℤ 看着是 ℕ 的「两倍再多一」,也照样可数——清单这么排:0, 1, −1, 2, −2, 3, −3, …,左右开弓地数,谁也不漏。(ℤ 同样先借用;严格构造它正是等价类机器的第一单生意。)

由此立刻多米诺:ℚ(有理数)也可数——每个有理数无非「一对整数」打了个商,塞进 ℤ×ℤ 的清单里挑着数即可。(细节将来 ℚ 建成后补严;眼下记住结论与方法:**清单化 = 可数**。)

到这里,你可能开始怀疑:难道所有无穷都一样多,「无穷」就是一个大箩筐?

## 第二记重锤:Cantor 定理——无穷分大小

不。Cantor 在 1891 年给出的论证,是整个数学里最漂亮的几页之一。先备一件小工具:集合 A 的**幂集**(power set) P(A),即 A 的全体子集组成的集合:P(A) := {X : X ⊆ A}。例如 P({1,2}) = {∅, {1}, {2}, {1,2}},四个元素。(「子集全体确实凑成一个合法的集合」——这我们约定接受,它是搭建集合宇宙的一条基本许可。)

**定理 (Cantor).** 对**任何**集合 A,不存在满射 f : A → P(A)。

**证明.** 任给一个函数 f : A → P(A)——它把每个元素 a 指派到一个子集 f(a) ⊆ A。要证 f 必不满,只需**亲手造出一个没人打中的靶点**(上上节「非满射」的标准任务:见证 + 一场 ∀ 的硬仗)。

造法是绝杀的一笔。对每个 a ∈ A 问一个问题:「**a 在不在自己被指派的那个子集里?**」把答案为「不在」的收拢起来:

D := { a ∈ A : a ∉ f(a) }。

(这是描述法的合法用法——从已有的 A 里切,不越雷池。)D 是 A 的子集,即 D ∈ P(A),是个正经靶点。往证:∀a ∈ A, f(a) ≠ D。

任取 a ∈ A,只问一件事:a ∈ D 吗?两种情形——

- 若 a ∈ D:按 D 的定义,a ∉ f(a)。于是 a 这个元素,在 D 里、不在 f(a) 里——两个集合的成员资格在 a 处不同,由[[fond_insiemi|外延原则]],f(a) ≠ D。
- 若 a ∉ D:按 D 的定义(取否定),a ∈ f(a)。于是 a 在 f(a) 里、不在 D 里——同理 f(a) ≠ D。

横竖 f(a) ≠ D,对一切 a 成立。D 无人打中,f 不满。∎

慢放一遍这支箭。D 是按 f 本人**量身定做**的反对者:它在每个坐标 a 上,都故意跟 f(a) 唱反调——你收 a,我就不收;你不收,我偏收。于是它跟清单上**每一项**至少有一处不同,自然不在清单上。这手法叫**对角线论证**(diagonal argument):沿着「a 对 f(a)」这条对角线逐点取反。你该觉得眼熟——罗素悖论里那个「不含自身的集合的集合」,用的是同一根针;只是罗素拿它扎破了朴素集合论,Cantor 拿它量出了无穷的层级。一针两用,这针法在数学和逻辑里还会反复显灵。

## 无穷的阶梯

Cantor 定理说:A 永远配不满 P(A)——而 A 塞**进** P(A) 倒容易(a ↦ {a} 是单射)。所以 P(A) 总是「严格更多」。把它迭代起来:

ℕ, P(ℕ), P(P(ℕ)), P(P(P(ℕ))), …

每一级都比上一级严格更大——**无穷不是一个箩筐,是一架没有顶的阶梯**。「可数」只是这架梯子的第一蹬;从 P(ℕ) 起,清单化就永远做不到了,这样的集合称**不可数**(uncountable)。预告一句:将来实数 ℝ 建成后,我们会证明 ℝ ≈ P(ℕ)——所以**实数不可数**,实数轴在「多少」的意义上就比自然数高一整级;届时还会用小数展开把对角线再扎一次,那是这一招的经典舞台。至于「两个集合互相单射进对方,是否就等势」(答案为是,Cantor–Schröder–Bernstein 定理),以及「梯级之间还有没有中间层」,都是这条线上的好戏,留待将来。

## 一道激活练习

把对角线握在手心。取 A = {1, 2, 3},定义 f : A → P(A) 为

f(1) = {2, 3},  f(2) = {2},  f(3) = ∅。

(a) 按定理证明里的配方,算出 D = {a ∈ A : a ∉ f(a)}。
(b) 逐一核对 D ≠ f(1)、D ≠ f(2)、D ≠ f(3),并对每个 a 指明「分歧发生在哪个元素上」。
(c) 这个 A 有 3 个元素,P(A) 有 8 个;f 当然不可能满。那 Cantor 定理对有限集说了什么**新**东西吗?想一想定理的分量到底压在哪类集合上。

## 参考解答

(建议先自己写完再看。)

**(a)** 逐个过堂:1 ∈ f(1) = {2,3}?否——1 入选 D。2 ∈ f(2) = {2}?是——2 落选。3 ∈ f(3) = ∅?否——3 入选。故 **D = {1, 3}**。

**(b)** D 与 f(1) = {2,3}:分歧在 1(1 ∈ D 而 1 ∉ f(1))——正是 1 自己的坐标。D 与 f(2) = {2}:分歧在 2(2 ∉ D 而 2 ∈ f(2))——正是 2 自己的坐标。D 与 f(3) = ∅:分歧在 3(3 ∈ D 而 3 ∉ f(3))——正是 3 自己的坐标。看见对角线了么:**D 与第 a 张牌的分歧,恰好都发生在 a 本人身上**——这正是「量身定做的反对者」的字面意思。

**(c)** 对有限集,「3 个元素打不满 8 个靶点」用数数就知道,定理确实不新。它的全部分量压在**无穷**集合上:那里「数数」失效,「部分≈整体」的怪事横行,谁也不敢担保 ℕ 打不满 P(ℕ)——而对角线给出了一个**对一切集合一视同仁**、根本不需要数数的论证。这也是个值得记住的品味:好定理的证明,常常在最危险的地方(无穷)和最平凡的地方(有限)用的是同一句话。
