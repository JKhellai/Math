## 取消最后的保留

环的公理对乘法宽容:不许诺幺元,不许诺逆元。这份宽容是为了容纳 ℤ 那样的世界——可我们最终要去的地方不止于此。在 ℚ 里、在 ℝ 里,除了零,**人人可除**:方程 a·x = b 只要 a ≠ 0 就该有解。把这最后两处保留也取消,让乘法享有与加法同等的待遇,就得到代数为数系准备的最完备的结构。

## 定义

**定义.** 一个**域**(field)是一个含幺交换环 (K, +, ·),满足:

- **(K1)** 1 ≠ 0;
- **(K2)** 每个 a ∈ K ∖ {0} 都有**乘法逆**(multiplicative inverse):∃b ∈ K, a·b = 1。该逆唯一(下详),记 a⁻¹。

习惯以字母 K 称呼域,源自德语 Körper(意语 corpo)——「体」;这门课沿用此字母。

定义里有两处限制,各有必要,逐一说明。

**为何把 0 排除在 K2 之外?** 因为不可能。设 0 竟有逆 b,则 1 = 0·b;可上一节定理 1 说 0·b = 0,于是 1 = 0,与 K1 相抵。**在任何 1 ≠ 0 的环里,0 都注定无逆**——K2 的措辞只是承认了这个事实。

**为何要求 1 ≠ 0?** 若 1 = 0,则对一切 a:a = a·1 = a·0 = 0(又是上一节定理 1),整个环坍缩成单点 {0}。这个「零环」并无矛盾,只是没有任何内容;K1 将它排除,纯属约定,为的是日后每条定理不必反复声明「设 K 非零环」。

把定义重新打包,会看见它的对称结构:**(K, +) 是交换群,(K ∖ {0}, ·) 也是交换群,分配律是两座群之间唯一的桥。** 加与乘,终于平起平坐。

> 这个表述顺带带来一批定理:[[alg_gruppi|群]]那一节的每条定理,如今**自动生效两次**——加法一次、乘法一次。逆元唯一?加法逆唯一(故 −a 合法),乘法逆也唯一(故 a⁻¹ 合法)。−(−a) = a?同理 (a⁻¹)⁻¹ = a。消去律?两种运算各得一条(乘法版限 a ≠ 0)。一次抽象证明,两处同时生效——公理化方法的力量,在域身上呈现得最为齐整。

## 除法诞生

仿照环里减法的出生,定义:对 b ≠ 0,

a / b := a · b⁻¹。

至此回望一条完整的弧线。在 ℕ 里,x + a = b 与 a·x = b 都未必有解;到了环,前者恒有唯一解 x = b − a;到了域,后者也尘埃落定——

**定理.** 设 K 是域,a, b ∈ K,a ≠ 0。则方程 a·x = b 在 K 中恰有一解,x = b/a。

**证明.** 存在性:a·(a⁻¹·b) = (a·a⁻¹)·b = 1·b = b。唯一性:设 a·x = b = a·x′,两边左乘 a⁻¹,得 x = x′。∎

(留意 b 可以为零:此时唯一解是 x = 0。受限的只是 a。)

## 最小的域

严格的样本,上一节其实已经造好,只待一次特化。取**单元素集** X = {p},看幂集环 (P(X), Δ, ∩):它只有两个元素——∅(零元)与 {p}(幺元)。非零元素仅 {p} 一个,而 {p} ∩ {p} = {p} = 幺元,即 {p} 的乘法逆是它自己:K2 成立;∅ ≠ {p},K1 成立。**它是一个域**——元素最少的域,记作 F₂。把两个元素改写作 0 与 1,运算表一览无余:

| + | 0 | 1 |
|---|---|---|
| **0** | 0 | 1 |
| **1** | 1 | 0 |

| · | 0 | 1 |
|---|---|---|
| **0** | 0 | 0 |
| **1** | 0 | 1 |

加法是「排斥或」,乘法是「且」——与逻辑联结词的对应一望便知。请注意表中关键的一格:**1 + 1 = 0**。域的公理对此毫无异议——没有任何一条公理许诺过 1 + 1 ≠ 0。这是一记提醒:「域」刻画的是**代数**上的完满,而我们心目中的数还有公理之外的另一半品性(大小、先后、正负),那一半要由别的公理另行供给,后文自有交代。

对照之下,上一节的反面教材立刻就位:X 含两个以上元素时,P(X) **不是**域——{1} ∩ {2} = ∅,零因子横行,而下面马上要证:域容不下零因子。同一族结构,X 的一字之差,域与非域立判,这比任何说教都更能划清边界。

至于将来:ℚ 与 ℝ 都将以域的身份建成;ℤ 则不是域——2 没有整数倒数——但它将是上一节定义的整环。「整环而非域」,正是 ℤ 在这套词汇里的身份,届时逐条核验。

## 域无零因子

**定理.** 域中无零因子:若 a·b = 0,则 a = 0 或 b = 0。

**证明.** 设 a·b = 0 且 a ≠ 0,往证 b = 0。a 非零,故有逆,于是

b = 1·b = (a⁻¹·a)·b = a⁻¹·(a·b) = a⁻¹·0 = 0,

末步是上一节的定理 1。∎

所以**每个域都是整环**。反过来不然——ℤ 将是见证(整环而非域),严格核验留待其建成之日。这条定理日常的用法,就是你解方程时的那句口头禅:「乘积为零,必有一个因子为零」——在域里,它是定理;在一般的环里(回想 P({1,2})),它会失效。口头禅也要看场合,场合由公理划定。

## 一道激活练习

设 K 是域,a, b, c, d ∈ K,b ≠ 0,d ≠ 0。证明小学的通分法则:

a/b + c/d = (a·d + b·c) / (b·d)。

(提示:三件事。先说明右边合法——b·d 为何非零?刚证的定理正候着。再把 / 全部按定义还原成乘逆。最后,(b·d)⁻¹ 等于什么?群那节「袜子与鞋」的定理在交换群里如何简化?)

## 参考解答

(建议先自己写完再看。)

**第一步:右边合法。** 须 b·d ≠ 0。反设 b·d = 0,由「域无零因子」得 b = 0 或 d = 0,均与假设相抵。故 b·d ≠ 0,(b·d)⁻¹ 存在,右边的除法有意义。

**第二步:化逆。** 在交换群 (K∖{0}, ·) 中,乘积之逆是逆之乘积((g∘f)⁻¹ = f⁻¹∘g⁻¹ 的此处版本;乘法交换,先后次序无关紧要):

(b·d)⁻¹ = b⁻¹·d⁻¹。

**第三步:展开右边,验证它等于左边。**

```
(a·d + b·c)·(b·d)⁻¹
= (a·d + b·c)·(b⁻¹·d⁻¹)          ⟨第二步⟩
= (a·d)·(b⁻¹·d⁻¹) + (b·c)·(b⁻¹·d⁻¹)   ⟨分配律⟩
= a·b⁻¹·(d·d⁻¹) + c·d⁻¹·(b·b⁻¹)        ⟨乘法交换律与结合律重排⟩
= a·b⁻¹·1 + c·d⁻¹·1
= a/b + c/d。                        ⟨除法的定义⟩
```

∎

回味:通分法则的每一步,都有一条公理或一条已证定理署名——分配律一次,逆元两次,「无零因子」在开演之前先保证了舞台合法。学校里它是背诵的程序;在这里,它是域公理的一则推论。今后在 ℚ、在 ℝ、在任何域里通分,你都知道自己凭的是什么。
