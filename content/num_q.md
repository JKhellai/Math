## 同样的问题,同样的方法

ℤ 已是有序的整环,可方程 2·x = 1 在其中无解——「一半」不在 ℤ 中。问题与当年 x + 3 = 1 在 ℕ 中的情形如出一辙:不是缺技巧,而是这个数系里**没有**那个对象。解决的办法也相同:不修补,扩张。让有序对 (a, b) 代表「a/b 这个尚不存在的商」,把「商相同」的对子粘成一类,类本身充当商。

只是这次,材料一上桌就带着一条限制:**第二坐标不得为零**。这并非谨慎过头——[[alg_campi|域]]那节证过,在任何 1 ≠ 0 的环里,0 注定无逆;「除以零」不是我们暂时不会,而是逻辑上无物可除。于是构造的底集取 ℤ × (ℤ ∖ {0}),这一限制从定义之初就写明。

## 「商相同」如何不用除法来说

熟悉的困难:「(a,b) 与 (c,d) 商相同」直觉上是 a/b = c/d,可除法尚未定义。移项(此刻仍在直觉层):a/b = c/d 「应当」等价于 a·d = c·b——右边只用 ℤ 的乘法,是已有的材料。扶正为定义:

**定义.** 在 ℤ × (ℤ∖{0}) 上,(a, b) ~ (c, d) :⟺ a·d = c·b。

**定理.** ~ 是等价关系。

**证明.** 自反:ab = ab。对称:等式两边互换。传递是要害:设 ad = cb …(i),cf = ed …(ii),其中 b, d, f ≠ 0,往证 af = eb。一条链算到底:

(a·f)·d = (a·d)·f = (c·b)·f = (c·f)·b = (e·d)·b = (e·b)·d,

五步只用交换结合重排与 (i)(ii)。两端是 (af)·d = (eb)·d,而 d ≠ 0——由[[num_z_ordine|整环消去律]],af = eb。∎

注意最后一步:消去 d,用的正是上一节证明的整环消去律。若底环含有零因子,消去律失效,传递性就无法证明——这正是构造 ℚ 之前必须先确立 ℤ 为整环的原因。同一套商集构造,对底下的结构是有要求的。

## 商集 ℚ 与分数记号

**定义.** ℚ := ( ℤ × (ℤ∖{0}) ) / ~。把 (a,b) 所在的类写作 **a/b**。

分数记号在此诞生——但请记住,它眼下只是**类的名字**;「a 除以 b」的字面意义,要等本节末尾才补上。先回应一个旧问题:[[fond_equivalenza|等价关系]]一节开篇问过——「1/2 与 2/4 写法不同,凭什么算同一个数?」——现在有了正式回答:**因为它们是同一个等价类**(1·4 = 2·2)。这个当初只能含糊回答的问题,如今有了严格的答案。

类的几何同样值得一看。把 (a, b) 画在「横轴 = 分母 b,纵轴 = 分子 a」的格点上:

一个类恰是**过原点的一条(去掉原点的)直线**——它的斜率,就是这个有理数:(2,1)、(4,2)、(−2,−1) 同在橙线上,正是 1/2 = 2/4 = (−1)/(−2)。竖直方向被排除——那是 b = 0 的方向。与上一处构造对照:ℤ 的类是一族**平行的对角线**,ℚ 的类是一束**过原点的直线**;同一套构造,两幅不同的几何。

## 运算的定义与良定义检验

直觉先行(通分与分子分母相乘,移项到只剩乘法):

a/b + c/d := (a·d + c·b)/(b·d),  (a/b)·(c/d) := (a·c)/(b·d)。

先验**合法性**:两条定义的结果都以 b·d 为第二坐标,它必须非零——b ≠ 0、d ≠ 0,整环无零因子,故 b·d ≠ 0。整环性质在此第二次使用:没有它,定义甚至给不出一个合法的对子。

再过**良定义**检验——按代表元定义,照例须验「换代表,结果类不变」。乘法的检验一行可了。换第一坐标:设 ab′ = a′b,往证 (ac, bd) ~ (a′c, b′d):

(a·c)·(b′·d) = (a·b′)·(c·d) = (a′·b)·(c·d) = (a′·c)·(b·d)。∎

第二坐标:乘法定义式在两坐标间对称(交换律),同一论证逐字适用;两步以 ~ 的传递性串接。加法的检验与此平行,工序你已在 ℤ 的构造里全程走过一遍——这次交给你,见文末练习。

## 核验身份:域

按域的定义验收。零元 0 := 0/1,幺元 1 := 1/1;(ℚ, +) 的结合、交换与 −(a/b) = (−a)/b(练习),乘法的结合、交换,分配律——全是「展开定义、用 ℤ 的定律在坐标上重排、收回类」的同款计算,路数在 ℤ 一节已示范,不再逐条铺陈。下面集中处理域独有的两条。

**K1(1 ≠ 0)**:1/1 = 0/1 将要求 1·1 = 0·1,即 ℤ 里 1 = 0,而 ℤ 含 ℕ 的忠实拷贝,1 ≠ 0。成立。

**K2(非零元可逆)**。先弄清「非零」在 ℚ 里说的是什么:

a/b = 0/1 ⟺ a·1 = 0·b ⟺ a = 0。

**一个分数为零,当且仅当分子为零**——分母没有发言权。于是设 a/b ≠ 0,即 a ≠ 0,定义

(a/b)⁻¹ := b/a。

**求逆,就是对调分子与分母。** 先验合法:对调后第二坐标是 a,恰因 a ≠ 0 而合规——「逆只对非零元素存在」这条域的规定,在构造里对应于:对调之后,「分母非零」的要求恰好落在 a 上。再验身份:(a/b)·(b/a) = (ab)/(ba) = (ab)/(ab),而 (ab)/(ab) ~ 1/1 ⟺ ab·1 = 1·ab,成立。∎

请把这一幕与 ℤ 对照:**取负是坐标对调(关于加法),求逆也是坐标对调(关于乘法)**。两次构造,各用一个对调补齐了原数系缺失的逆元——同一个动作,用在两种运算上。

**结论:(ℚ, +, ·) 是域。** 最初的方程就此解决:2·x = 1 的解是 x = 1/2,验算 (2/1)·(1/2) = (2·1)/(1·2) = 2/2 ~ 1/1。每个非零搭配都能除——域那节练习里证下的通分法则、唯一解定理,从今天起在 ℚ 上句句生效。

## ℤ 的嵌入与记号的字面化

照例,新数系须包含旧数系。定义 ι : ℤ → ℚ,a ↦ a/1。单射:a/1 = c/1 ⟺ a·1 = c·1 ⟺ a = c。保加法:ι(a) + ι(c) = (a·1 + c·1)/(1·1) = (a+c)/1 = ι(a+c)。保乘法同理。于是把 a 与 a/1 认同,[[num_z|ℤ]] 连同它体内的 ℕ 一并整体迁入。

记号的字面意义随之确立:对任意 b ≠ 0,

ι(a) · ι(b)⁻¹ = (a/1)·(1/b) = (a·1)/(1·b) = a/b。

**a/b 从此字面上就是「a 除以 b」**——当初只是类名的斜杠记号,如今是 ℚ 内部真正的除法。与 ℤ 一节「[(a,b)] = a − b 字面成立」相呼应:每次构造的最后一步,都是让借用的直觉记号获得严格定义。

随后照例:**造出来 → 验证性质 → 封装遗忘**。自下一节起,我们使用 ℚ 只凭其性质——一个域,内含 ℤ(进而 ℕ)的忠实拷贝——不再翻看对子与斜线。

## 回望:构造数系的一般流程

两个新数系建成,流程已可提炼:先**用旧运算刻画「相同」**(差相同、商相同——总是一次移项);再**验等价**,传递性总是要紧处,总需要一条消去律;然后**商出新数系**,在代表元上**定义运算**并逐条完成**良定义**的检验;最后**核验代数身份、嵌入旧数系、让记号字面化**。这套流程对任何整环逐字适用,产物称该整环的**分式域**(field of fractions)——ℚ 就是 ℤ 的分式域;一般理论留待代数主干。你掌握的不是两个孤立的构造,而是一套可以重复使用的流程。

## 一道激活练习

(a) 补全加法的良定义检验(换第一坐标的情形):设 b, b′, d ≠ 0,a·b′ = a′·b。证明

(a·d + c·b, b·d) ~ (a′·d + c·b′, b′·d)。

并说明换第二坐标的情形为何不必另证。

(b) 验证加法逆元:(a/b) + ((−a)/b) = 0/1。

(提示:(a) 把 ~ 的定义展开成一条等式,两边各自分配展开,逐项比对——其中一对项恰差一个 a·b′ = a′·b,另一对靠交换律。(b) 通分相加后,分子里有一处可以提取公因子;随后环的定理「0·x = 0」收尾。)

## 参考解答

(建议先自己写完再看。)

**(a)** 按定义,须证 (a·d + c·b)·(b′·d) = (a′·d + c·b′)·(b·d)。两边分配展开:

左 = a·d·b′·d + c·b·b′·d = (a·b′)·d² + c·b·b′·d,
右 = a′·d·b·d + c·b′·b·d = (a′·b)·d² + c·b·b′·d,

第二项两边相同(交换律),第一项由假设 a·b′ = a′·b 相等。故两边相等。∎

换第二坐标不必另证:加法定义式在 (a,b) ↔ (c,d) 互换下**对称**(a·d + c·b = c·b + a·d,b·d = d·b),把「换第二坐标」经一次互换化归为「换第一坐标」;最后两步以 ~ 的传递性串接,与乘法一节的安排相同。∎

**(b)** 按定义,(a/b) + ((−a)/b) = ( a·b + (−a)·b , b·b )。分子提取公因子(分配律倒用):

a·b + (−a)·b = ( a + (−a) )·b = 0·b = 0,

末步是环的定理 1。于是和为 0/(b·b)。最后 0/(b·b) ~ 0/1 ⟺ 0·1 = 0·(b·b) ⟺ 0 = 0。∎(顺带:b·b ≠ 0 由无零因子保证,这个对子自始合法。)

回顾 (b) 的三步——分配律、0·x = 0、按定义核对等价——没有一步是 ℚ 的新发明,全是 ℤ 与环的已有定理在新坐标上的组合。构造的省力之处正在于此:新数系的大半验证,由旧数系的定理代劳。

## 往前看一步

ℚ 是域:加减乘除四则齐备,代数意义上再无残缺。可数,还不只是代数——2 比 1/2 大、−3 在 0 左边,这些话在 ℚ 里尚无定义:ℤ 的序还没有传过来。移植的办法你已在 ℤ 见过一回,但这次有个新情况:分母可正可负,「a/b 是否为正」不能直接读分子——得先把代表整理到**分母为正**的一侧。序立起来之后,ℚ 将获得一个完整的身份:**有序域**——「平方非负」一类定理的自然归属。这就是 [[num_q_ordine|ℚ 的序与有序域身份]] 的内容。
