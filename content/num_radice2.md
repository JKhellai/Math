## 一桩两千年前的丑闻

边长为 1 的正方形,对角线有多长?几何直觉答曰:某个数 d,满足 d² = 1² + 1² = 2。

(此处的「勾股」只作引子——它是几何主干的定理,严格证明留待那一支开张;本节要证的是一个**纯算术**命题,不依赖任何几何。)

毕达哥拉斯学派相信「万物皆数」,而他们说的数,是整数与整数之比——用我们的话说,ℚ。传说中,希帕索斯发现这条对角线**不是**任何整数之比,搅动了整个学派的根基。上一节刚刚盘点过 ℚ 的家底:域、全序、相容、处处稠密——以一切肉眼可见的标准,这条数轴天衣无缝。今天要严格证明的是:

**定理.** 不存在 x ∈ ℚ,使得 x² = 2。

(标题里的「√2」眼下只是修辞:我们要证的恰是这个记号在 ℚ 中**无物可指**。它将来在 ℝ 中获得所指。)

## 先取回一件寄存的工具:良序

序那节末尾有句话:「任何非空子集都有最小元……表述与证明留待用到之日。」**就是今天**——本节的证明要在一族分母中取「最小的」,没有良序,这一步无处落脚。而良序的证明,又需要先看清 ℕ 的一个本性:它是**离散**的。

**引理(离散性).** ∀m, n ∈ ℕ:m < S(n) ⟺ m ≤ n。等价的另一面:n < m ⟺ S(n) ≤ m。**n 与 S(n) 之间,没有自然数。**

**证明.** 「⟸」设 m ≤ n。又 n < S(n)(见证 1;且 n ≠ S(n)——这正是 Peano 那节练习的定理),传递得 m ≤ S(n);若 m = S(n),则 S(n) ≤ n,与 n ≤ S(n) 反对称夹出 n = S(n),已被排除。故 m < S(n)。

「⟹」设 m < S(n),取见证 k:m + k = S(n),且 k ≠ 0(否则 m = S(n))。由「零或后继」,k = S(j),于是 S(m + j) = m + S(j) = S(n),由 P3(S 单射)剥壳:m + j = n,即 m ≤ n。另一面同理(把见证拆成 S(j),重排出 S(n) + j = m)。∎

**定理(良序原理 well-ordering principle).** ℕ 的任何非空子集 M 都有**最小元**:存在 μ ∈ M,使 ∀m ∈ M, μ ≤ m。

**证明.** 证逆否:设 M 没有最小元,往证 M = ∅。对谓词 P(n) :≡「∀m ≤ n,m ∉ M」做归纳。

奠基 P(0):凡 m ≤ 0,又恒有 0 ≤ m,反对称得 m = 0;而若 0 ∈ M,则 0 就是 M 的最小元(0 不超过一切),与假设相抵,故 0 ∉ M。

归纳步:设 P(n)。取 m ≤ S(n):若 m ≤ n,由 P(n) 知 m ∉ M;否则由全序性 n < m,配 m ≤ S(n) 与离散性,夹出 m = S(n)。于是只剩一问:S(n) 是否在 M?设 S(n) ∈ M。任取 m′ ∈ M,由 P(n) 知 ¬(m′ ≤ n),即 n < m′,离散性给 S(n) ≤ m′——这说明 S(n) 是 M 的最小元,与假设相抵。故 S(n) ∉ M,P(S(n)) 成立。

由归纳原理,∀n P(n);而每个 n 满足 n ≤ n,故每个 n 都不在 M 中:M = ∅。∎

归纳与良序,自此互为表里:归纳说「步步可达即全部」,良序说「想找最小的,总找得到」。(反向推导——由良序出发重获归纳——以及它的孪生「强归纳法」,留待各自用到之日。)

## 奇与偶

主定理的第二件装备是奇偶性的严格处理。称 n ∈ ℕ 为**偶数**(even),若 ∃k, n = 2k;为**奇数**(odd),若 ∃k, n = 2k + 1(其中 2 := S(S(0)))。

**引理(二歧).** 每个自然数是偶数或奇数,且不能两者皆是。

**证明.** *至少居其一*,对 n 归纳:0 = 2·0 是偶数;若 n = 2k,则 S(n) = n + 1 = 2k + 1 是奇数;若 n = 2k + 1,则 S(n) = 2k + 2 = 2(k+1) 是偶数(分配律)。

*至多居其一*:设竟有 2a = 2b + 1。由三歧性分两路。若 a ≤ b:由乘法相容 2a ≤ 2b,而 2b < 2b + 1(见证 1;若相等则消去得 0 = 1 = S(0),抵触 P4),串起来 2a < 2a,荒谬(< 不自反)。若 b < a:离散性给 b + 1 ≤ a,乘 2 得 2b + 2 ≤ 2a = 2b + 1,消去 2b 得 2 ≤ 1;而 1 < 2(同前法核验),反对称夹出 1 = 2,消去得 0 = 1,再抵 P4。两路皆死。∎

(留意离散性引理已经第二次上岗——它是本节真正的劳模。)

**引理(关键).** 奇数的平方是奇数;从而,**n² 为偶 ⟹ n 为偶**。

**证明.** 设 n = 2k + 1。纯环计算:

n² = (2k+1)(2k+1) = 2k(2k+1) + (2k+1) = (4k² + 2k) + (2k + 1) = 2(2k² + 2k) + 1,

是奇数。于是若 n² 为偶而 n 不偶:由二歧之「至少」,n 为奇;刚证 n² 为奇;再由二歧之「至多」,n² 不偶——矛盾。∎

请看清这条推论同时动用了二歧的**两半**:「至少」把「非偶」翻译成「奇」,「至多」禁止 n² 脚踏两船。定义里每个字都不是闲笔。

## 主定理

**定理.** 不存在 x ∈ ℚ 使 x² = 2。

**证明.** 反设存在。先整顿对象。x ≠ 0(否则 x² = 0,而 0 ≠ 2,因 0 < 1 < 2)。又 x² = (−x)²(负负得正),故由三歧性,不妨设 x > 0。取分母为正的代表 x = p/q,q ≥ 1;x > 0 配正性刻画给 p ≥ 1。至此 p, q 落在 ℕ 中。

让良序出场。令

D := { q ∈ ℕ : q ≥ 1,且存在 p ∈ ℕ 使 p/q = x }。

刚才的整顿说明 D 非空;由良序原理,D 有最小元 q₀,配套分子 p₀。

把假设 x² = 2 翻译到整数层:(p₀/q₀)·(p₀/q₀) = 2/1,按 ℚ 的乘法与相等定义,即

p₀² = 2·q₀²。

右边是偶数,故 p₀² 为偶,由关键引理 **p₀ 为偶**:p₀ = 2k。代回:4k² = 2q₀²,两边消去 2(乘法消去律,2 ≠ 0):

q₀² = 2·k²。

同一引理再用一次:**q₀ 为偶**,q₀ = 2j。此时 j ≠ 0(否则 q₀ = 0),且 j < q₀(因 q₀ = 2j = j + j ≥ j + 1 > j)。而 (k, j) 与 (p₀, q₀) 等价:k·q₀ = k·2j = 2k·j = p₀·j。所以 k/j = x,且 j ∈ D,j < q₀——**与 q₀ 的最小性相抵**。反设崩塌。∎

回看这场下降:从任何一个代表出发,假设都迫使分子分母**双双减半**,半而又半,永无止境——可良序原理早已立法:**ℕ 中不存在无穷下降**。古希腊人称这手法为无穷递降;在我们的语言里,它就是良序原理的反面拳。证明里每件工具都有名有姓:良序取最小、关键引理两次开锋、乘法消去律拆 2、等价关系收尾——没有一步是「显然」。

## 这意味着什么

把定理放回上一节的盘点旁边,才见其惊心:ℚ 处处稠密——任何放大倍数下都看不到空隙——**然而它有洞**。集合 {r ∈ ℚ : r > 0, r² < 2} 里的数可以把「平方」推到无限接近 2,却没有任何成员、也没有任何 ℚ 中的「天花板」恰好站在 2 的正上方(这句话的严格版本,正是下一节的任务)。稠密与无洞,原来是两回事——肉眼看不见的空隙,要用更精细的语言才能说出口。

至于那条对角线:它的「长度」要等几何主干立起勾股、ℝ 建成赋予 √2 所指之后,才能严格地谈。今天确立的是纯算术的事实;几何的兑现,各归其位,来日方长。

## 一道激活练习

(a) 仿照正文,证明:**不存在 x ∈ ℚ 使 x² = 3。**(提示:奇偶二歧换成**三歧**——每个自然数形如 3k、3k+1、3k+2 之一;算出三种形状的平方除以 3 的余数,提炼出「3 整除 n² ⟹ 3 整除 n」,随后下降照旧。三分的「至少居其一」用归纳,「至多」与正文同法。)

(b) 同样的论证若用于 x² = 4,会在哪一步失效?为什么它**理应**失效?

## 参考解答

(建议先自己写完再看。)

**(a)** *三歧之「至少」*,对 n 归纳:0 = 3·0;若 n = 3k,则 n+1 = 3k+1;若 n = 3k+1,则 n+1 = 3k+2;若 n = 3k+2,则 n+1 = 3k+3 = 3(k+1)。三种形状轮转,每步落回其一。「至多」(两两互斥)与正文奇偶互斥同法:假设两形相等,用离散性与消去逼出 0 = 1 或 1 = 2 一类的抵触,从略。

*关键引理*:三种形状的平方——

3k 型:(3k)² = 3·(3k²);
3k+1 型:(3k+1)² = 9k² + 6k + 1 = 3(3k² + 2k) + 1;
3k+2 型:(3k+2)² = 9k² + 12k + 4 = 3(3k² + 4k + 1) + 1。

唯有 3k 型的平方仍是 3 的倍数,故 **3 | n² ⟹ 3 | n**(用三歧两半,与正文同构)。

*下降*:反设 x² = 3,同法整顿出 p₀/q₀ = x,p₀, q₀ ∈ ℕ,q₀ 取 D 中最小。翻译:p₀² = 3q₀²,故 3 | p₀,p₀ = 3k;代回 9k² = 3q₀²,消去 3:q₀² = 3k²,故 3 | q₀,q₀ = 3j;j ≠ 0,且 j < q₀(q₀ = 3j = j + 2j ≥ j + 2 > j),而 k·q₀ = 3kj = p₀·j 给出 k/j = x。最小性被破。∎

**(b)** 失效点在**关键引理的对应物**:「4 | n² ⟹ 4 | n」是假的——n = 2 即反例(4 整除 4,而 4 不整除 2)。若改沿奇偶路线推进:p₀² = 4q₀² 只能给出 p₀ 为偶,p₀ = 2k,代回得 4k² = 4q₀²,消去后 **k² = q₀²**——分母没有变小,下降的链条在此断裂,再无矛盾可榨。它理应断裂:x = 2 明明白白地是 ℚ 的成员,2² = 4。**一个方法的失效点,与一个对象的存在点恰好重合**——这不是巧合,而是好证明的试金石:它该通则通,该断则断。

## 往前看一步

裂缝定位完毕。但「洞」眼下还只是比喻——ℚ 的数轴上既然处处稠密,「缺了一个点」这句话究竟在**说什么**?要把它说严格,需要一套新的语言:给一个集合谈它的**上界**,再问诸多上界里有没有**最小的一个**。有了这套语言,今天的定理将改写成一句更深的话:在 ℚ 里,有界的集合可以没有最小的天花板。这套语言,以及它所定义的那条把 ℚ 与 ℝ 一刀分开的性质,就是 [[num_estremo_superiore|上确界]] 的内容。
