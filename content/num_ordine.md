## 开工前的最后一件装备

ℤ 的图纸已经定了:在 ℕ × ℕ 上铺一张等价关系,商出「差」。可机器一开动,第一道工序就要用到一件 ℕ 还没有的东西——**消去律**:从 m + k = n + k 推出 m = n。没有它,那张等价关系连传递性都验不过,机器空转。此外还有一桩悬置已久的事:我们整天说 2 比 5 小,可 ℕ 上的「小于等于」至今没有定义。这一节把两件事一并办妥——而且你会看到,它们出自同一口井:序,本来就是用加法挖出来的。

## 消去律:没有逆元,也能消去

先回想[[alg_gruppi|群]]那节的一句旁白:群里的消去律是三行公理推演(左乘逆元即得),而「在 ℕ 里它也成立,但需要对 ℕ 的具体构造做一番归纳」。现在交付那番归纳。

**定理(加法消去律 cancellation law).** ∀m, n, k ∈ ℕ:m + k = n + k ⟹ m = n。

**证明.** 固定 m、n,对 k 归纳。

奠基 k = 0:m + 0 = n + 0,由 A1 即 m = n。

归纳步:设结论对 k 成立。若 m + S(k) = n + S(k),由 A2 得 S(m + k) = S(n + k);S 是[[num_n|单射]](公理 P3!),剥去外壳得 m + k = n + k;由归纳假设,m = n。∎

请品味两件事。其一,撑起每一步「剥壳」的是 P3——后继不撞车,所以加法的历史可以一步步倒查回去。其二,把这份证明与群里那三行并排:**群靠逆元消去,ℕ 靠归纳消去**。逆元是更省力的高地,但没有逆元的世界未必没有消去——只是要换一种力气。这个对照,正是「公理推演」与「具体构造」两种工作方式的缩影。

## 序:可达即不超过

直觉里「m 不超过 n」,就是「从 m 出发,加上某段路程能到 n」。把直觉原样写下:

**定义.** m ≤ n :⟺ ∃k ∈ ℕ, m + k = n。又定义 m < n :⟺ m ≤ n ∧ m ≠ n。

那个 k 就是[[fond_quantificatori|存在命题]]的见证——「距离」。例如 2 ≤ 5,见证 k = 3(因 2 + 3 = 5);而 0 ≤ m 对一切 m 成立,见证是 m 自己(0 + m = m,算术那节的引理 A)。

这是本课程的数轴第一次登场——它此刻只是一串离散的脚印,但「序 = 沿轴可达」这幅图景,会一路陪我们走到 ℝ。

## 两条小引理

后面的证明反复需要两件小工具,先备好。

**引理 1(零或后继).** ∀n ∈ ℕ:n = 0 或 ∃j, n = S(j)。

**证明.** 对 n 归纳。奠基:n = 0,左支成立。归纳步:S(n) 自己就是个后继(取 j = n),右支成立。∎

(留意这个归纳步**根本没用归纳假设**——结论自己就站住了。归纳法并不强迫你每次都借力,它只要求每一环确实接上。)

**引理 2(和为零).** k + l = 0 ⟹ k = 0 且 l = 0。

**证明.** 由引理 1,对 l 分情形。若 l = S(j),则 k + S(j) = S(k + j) = 0,与公理 P4(0 不是后继)相抵,此情形不会发生。故 l = 0;于是 k = k + 0 = 0。∎

## 序的品性

现在逐条验明 ≤ 的身份。回想[[fond_relazioni|关系]]那节末尾按下的那句话:≤ 自反、传递、却不对称,「是另一个重要物种的代表」。现在正式引见这个物种。先把 ≤ 缺的那块拼上——它虽不对称,却有一条方向相反的纪律:

**反对称(antisymmetric)**:∀x, y ( x R y ∧ y R x ⟹ x = y )——双向可达,必是同一处。

自反 + 反对称 + 传递,合称**序关系**(order relation),也叫**偏序**(partial order);若再添「任两元素皆可比较」,称**全序**(total order)。

**定理.** ℕ 上的 ≤ 是全序。

逐条验收:

**自反**:m + 0 = m,见证 0。

**传递**:设 m ≤ n、n ≤ p,见证分别为 k、l:m + k = n,n + l = p。于是 m + (k + l) = (m + k) + l = n + l = p(中间一步是你亲手证的结合律),见证 k + l。距离会叠加——传递性不过是这句话。

**反对称**:设 m ≤ n 且 n ≤ m,见证 k、l:m + k = n,n + l = m。代入得 m + (k + l) = m = m + 0;由交换律换边、消去律消去 m,得 k + l = 0;引理 2 断言 k = l = 0,故 n = m + 0 = m。∎(消去律的第一次上岗——没有它,反对称性就证不动。)

**全序性**:∀m, n:m ≤ n 或 n ≤ m。固定 m,对 n 归纳。

奠基 n = 0:0 + m = m,故 0 ≤ m,右支成立。

归纳步:设 m ≤ n 或 n ≤ m,往证 m ≤ S(n) 或 S(n) ≤ m。分情形。

- 情形一,m ≤ n:又有 n ≤ S(n)(见证 1,因 n + 1 = S(n),算术那节顺手证下的小定理),由传递性 m ≤ S(n)。
- 情形二,n ≤ m,取见证 k,n + k = m。由引理 1 再分:
  - k = 0:则 m = n,而 n ≤ S(n),故 m ≤ S(n)。
  - k = S(j):则 m = n + S(j) = S(n + j) = S(n) + j(末步是算术那节的引理 B),故 S(n) ≤ m,见证 j。

各情形俱得结论。∎

于是任两个自然数都能分出高下;配上反对称,便有**三歧性**(trichotomy):m < n、m = n、n < m 恰居其一(核验留给乐意动笔的读者)。顺带一份对照,见识「偏序而非全序」长什么样:幂集上的包含 ⊆——自反与传递在集合那节早已是日用品,而它的反对称性正是你证集合相等时的「双向夹」(A ⊆ B 且 B ⊆ A ⟹ A = B)!但它不是全序:{1} 与 {2} 谁也不包含谁,无从比较。「序」这个物种,远比数轴上这一条来得多样。

ℕ 的序还有一条更深的性质——任何非空子集都有最小元(**良序** well-ordering),它与归纳原理实为一体之两面;表述与证明留待用到之日。

## 序与运算相容

将来 ℤ、ℚ、ℝ 的序都要「尊重运算」,这份默契在 ℕ 这里已经成立,各一行:

**加法相容**:m ≤ n ⟹ m + p ≤ n + p。(见证不变:m + k = n,则 (m+p) + k = (m+k) + p = n + p,中间用交换与结合律重排。)

**乘法相容**:m ≤ n ⟹ m·p ≤ n·p。(见证翻倍:n = m + k,则 n·p = (m+k)·p = m·p + k·p——分配律——见证 k·p。)

这两条此刻是定理;将来在**有序域**的公理清单上,它们将以公理的身份再次现身——同一句话,在不同的世界里身份不同,这正是「具体构造」与「公理刻画」之间的常态往返。

## 一道激活练习

设 m ≤ n。证明:满足 m + k = n 的见证 k **唯一**。

由此可以放心地定义 **n − m** 为这个唯一的 k——ℕ 至此获得一个**部分的**减法,只在 m ≤ n 时有定义。(提示:两个见证摆在一起,你有一条刚出炉的定理正等着消去什么。)

## 参考解答

(建议先自己写完再看。)

设 k、k′ 都是见证:m + k = n = m + k′。由交换律改写为 k + m = k′ + m,由消去律消去 m,得 k = k′。∎

照旧体会一句:**唯一性使记号合法**——正因见证只有一个,「n − m」三个字才指称确定的对象;这与当初 f(a)、f⁻¹ 获得记号资格的方式一脉相承(背后都是 ∃!)。但也请看清这部减法的残缺:它只对 m ≤ n 开放,1 − 3 依旧无处安身。修补不是办法——下一站,我们不修补,我们扩张。

## 往前看一步

装备清点完毕:加法、乘法及其定律,消去律,全序。ℕ 能给的都给了,而 x + 3 = 1 依旧无解——问题不在工具,在世界本身太小。出路在[[fond_equivalenza|等价类]]那节早已预告:到 ℕ × ℕ 上去,让每个有序对 (a, b) 代表「a − b 这个尚不存在的差」,把「差相同」的对子粘成一类,商出一个崭新的世界。机器的每个零件——等价关系要靠消去律验传递,商集上的运算要靠「与代表元无关」站稳——都已在你手中。开工:[[num_z|ℤ 的构造]]。
