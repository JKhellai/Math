## 从自由到纪律

[[fond_relazioni|关系]]给了我们「对应」，但那是一种完全放任的对应：一个 a 可以指向好几个 b，也可以谁都不指。可数学里最当家的对应是另一种气质——「**给我一个输入，我交还你一个明确的输出**」：给一个数，交还它的平方；给一个三角形，交还它的面积。这里的要害是**确定性**：同一个输入，不许时而这样、时而那样，也不许耸耸肩说「没有输出」。把这条纪律施加到关系上，得到的就是函数。

## 先补一件量词的零件：「恰好一个」

纪律说的是「每个输入**恰好**对应一个输出」——不多（唯一性）、不少（存在性）。「恰好一个」值得有自己的记号。在 [[fond_quantificatori|量词]]的语言里定义：

∃!x P(x)  :⟺  ∃x ( P(x) ∧ ∀y ( P(y) ⟹ y = x ) )，

读「**恰好存在一个** x 使 P(x)」(there exists a unique)。拆开看正是两件事打包：前半个 ∃x P(x) 给**存在**；后半句「凡满足 P 的都等于这个 x」给**唯一**。以后任何「存在且唯一」式的断言（数学里许多重要定理都是这种形状），证明照例分两段：先造一个、再证只此一个。

## 定义

设 A、B 是集合。A 到 B 的一个**函数**(function)（同义词：**映射**(map)），是一个关系 f ⊆ A × B，满足：

∀a ∈ A, ∃!b ∈ B : (a, b) ∈ f。

一句话：A 的每个元素，**恰好**被指派 B 里的一个元素。由于这个 b 被 a 唯一确定，给它专属记号 **f(a)**，叫 f 在 a 处的**值**(value)。整套数据写作

f : A → B，  a ↦ f(a)，

其中 A 叫**定义域**(domain)，B 叫**到达域**(codomain)；箭头 → 连接两个集合，↦ 描述单个元素的去向，两个箭头别混用。

留意我们没有引入任何新本体：函数「就是」一种特殊的关系，而关系「就是」笛卡尔积的子集——一路还原到底，函数是个集合。你熟悉的「函数图像」在这里翻转了身份：**f 不是『有』一个图像，f 本身就是它的图像**——那堆有序对 {(a, f(a)) : a ∈ A} 不是别的，正是 f 自己。

两个函数何时**相等**？数据有三件：定义域、到达域、逐点的值。三件都同才算同：f = g 当且仅当定义域相同、到达域相同，且 ∀a, f(a) = g(a)。前两件不是迂腐——同一条「求平方」规则，配上到达域 ℝ 和配上到达域 [0, ∞)，是**两个不同的函数**；这个区分眼下显得吹毛求疵，到下一节谈「满射」时将变得关键。（ℝ 照例先按熟悉的方式借用，严格构造在后头。）

## 像：到达域里真正被打中的部分

到达域 B 是**宣称**的弹着范围，实际被打中的未必是全部。把真正取到的值收起来：

im f := { f(a) : a ∈ A } ⊆ B，

叫 f 的**像**(image)。例如「求平方」ℝ → ℝ 的像是 [0, ∞)：负数在到达域里坐着，却永远没人打中它。**像 ⊆ 到达域，且一般是真包含**——这两个集合的张力，正是下一节的主角之一。

## 例子与反例

**是函数的：**

- **恒等函数**(identity) id_A : A → A, a ↦ a。最朴素也最重要——以后谈「逆」时它是基准。
- **常函数**(constant)：取定 c ∈ B，f : A → B, a ↦ c。所有输入打同一个点。「多对一」完全合法——纪律只管「一个输入不许多个输出」，不管「多个输入共用一个输出」。
- f : ℝ → ℝ, x ↦ x²。每个实数恰有一个平方。

**不是函数的（病各不相同）：**

- R = {(x, y) ∈ ℝ×ℝ : x = y²}，当作 ℝ 到 ℝ 的对应（输入 x、输出 y）。**两种病都犯**：x = −1 时没有任何 y（存在性失败）；x = 4 时 y = 2 和 y = −2 都行（唯一性失败）。
- 「把每个人对应到他的孩子」：无子女者处存在性失败，多子女者处唯一性失败。日常语言里大量「对应」经不起这两条盘问——这正是函数定义的筛选力。

## 一句要紧的眺望：数学的对象，将批量地「是」函数

定义虽朴素，辐射面大得惊人，先指给你三处：

- **数列**就是函数：一个实数列无非是某个 f : ℕ → ℝ（第 n 项 = f(n)）。将来分析整层楼——极限、级数——的地基语言就是它。
- **运算**就是函数：加法无非是 + : ℕ×ℕ → ℕ，(a,b) ↦ a+b。代数那一支研究的「带运算的集合」，运算二字的严格身份就是这种函数。
- 上一节的商集构造里还有一个值得点名的函数：把每个元素送到它的等价类，a ↦ [a]，这是一个函数 A → A/~（叫**典范投影**）。以后构造 ℤ 时会直接用到它。

## 一道激活练习

A = {1, 2, 3}，B = {a, b}。下列 A×B 的子集，哪些是 A 到 B 的函数？是的写出 f(1), f(2), f(3) 与像 im f；不是的指明哪一条纪律在哪个元素处失败。

(1) R₁ = {(1,a), (2,a), (3,b)}
(2) R₂ = {(1,a), (1,b), (2,a), (3,b)}
(3) R₃ = {(1,a), (3,b)}
(4) R₄ = {(1,b), (2,b), (3,b)}

（提示：对**每个** a ∈ A 逐一盘问两件事——有没有箭？只有一支吗？）

## 参考解答

（建议先自己写完再看。）思路：定义就是检查清单——∀a ∈ A，先验存在（至少一对 (a, ·)），再验唯一（至多一对）。逐个过堂：

**(1) 是函数。** 1、2、3 各恰好出现一次作第一分量。f(1)=a, f(2)=a, f(3)=b；im R₁ = {a, b}。1 和 2 共用输出 a 完全合法——多对一不犯规。

**(2) 不是。** 元素 1 处**唯一性失败**：(1,a) 与 (1,b) 同时在场，f(1) 无从谈起。（2、3 处都干净也救不了它——纪律是 ∀，一处失守即全盘否决。）

**(3) 不是。** 元素 2 处**存在性失败**：没有任何形如 (2, ·) 的对。它倒是个正经的关系——只是不配叫函数。

**(4) 是函数。** 三个输入各恰好一箭，全打向 b——这是常函数。f(1)=f(2)=f(3)=b；im R₄ = {b}。注意 **im R₄ = {b} ⊊ B**：到达域里 a 没人打中。「像真包含于到达域」在这里第一次具体现身。

回味一句：(2) 和 (3) 病灶相反——一个箭太多、一个箭太少——恰好对应 ∃! 里打包的两半（∃ 与唯一性）。检查任何「对应」是不是函数，永远是这两问。

## 往前看一步

至此地基的四件套——逻辑、集合、关系、函数——全部到位。而函数自己的故事才刚开头：(4) 里那种「到达域没被打满」的函数，与「多对一」的函数，分别欠缺两种宝贵的品质；两种品质凑齐的函数，能把两个集合「完美配对」，进而让我们第一次能严格地谈论「这两个集合一样多」——哪怕它们都是无穷的。这就是接下来 [[fond_iniettive_suriettive|单射与满射]] 的戏。
