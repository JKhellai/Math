## 两种缺憾，两种品质

上一节末尾验明：函数的纪律只管「一个输入恰好一支箭」，并不禁止两件事——**几个输入共用一个靶点**（常函数是极端），以及**到达域里有靶点无人问津**（im f ⊊ B）。这两件事不犯规，却各是一种缺憾：前者意味着从输出**回推不出**输入是谁；后者意味着到达域**虚报**了弹着范围。把这两个缺憾分别堵上，就得到函数的两种品质；两种全占的函数，将做到一件了不起的事——在两个集合之间**完美配对**。

## 单射：输出认得出输入

f : A → B 是**单射**(injective)，若不同输入必得不同输出。直接写就是 ∀x, x′ ∈ A ( x ≠ x′ ⟹ f(x) ≠ f(x′) )。但实际干活时，几乎总用它的[[fond_proposizioni|逆否]]形式：

∀x, x′ ∈ A ( f(x) = f(x′) ⟹ x = x′ )。

两种说法逻辑等价（逆否命题与原命题同真假——第一节已证的事实），但**好用程度天差地别**：原始形式叫你从「x ≠ x′」这种否定信息出发去推另一个否定——不等式没法代入、没法消去，无从下手；逆否形式给你一个**等式** f(x) = f(x′) 当起点——等式可以两边同加同乘、层层剥洋葱。于是**证单射的标准开场白**永远是：「设 f(x) = f(x′)，往证 x = x′」。

单射也叫「一对一」(one-to-one)——每个被打中的靶点恰被一支箭打中，从输出能**唯一回溯**输入。

## 满射：到达域不掺水

f : A → B 是**满射**(surjective)，若到达域里每个元素都真的被打中：

∀b ∈ B, ∃a ∈ A : f(a) = b。

等价说法：im f = B——「像」与「到达域」这对上一节区分的概念在此合龙。满射也叫「映上」(onto)。

证明的形状由定义里的 ∀∃ 决定：**任给** b ∈ B，要**找出**（造出）一个 a 使 f(a) = b——这是[[fond_quantificatori|存在命题]]，给出见证即可，通常就是「解方程 f(a) = b，把 a 用 b 表示出来，再代回验证」。

立刻看清一件事：**满射与否，根本不是 f 的「规则」单方面能决定的——它由规则与到达域共同决定**。同一条规则，到达域写大一号就不满，裁齐了就满。这正是上一节坚持「到达域是函数身份的一部分」的落实。

## 一条规则，四种身份

把这层意思推到极致。看「求平方」这一条规则，配四种(定义域, 到达域)：

| | 函数 | 单射? | 满射? | 病灶 / 解法 |
|---|---|---|---|---|
| ① | ℝ → ℝ, x ↦ x² | 否 | 否 | f(−1)=f(1)=1 杀单射；−1 无人打中杀满射 |
| ② | [0,∞) → ℝ, x ↦ x² | 是 | 否 | 砍掉负半轴，「±x 同平方」的问题消失了；但 −1 仍空置 |
| ③ | ℝ → [0,∞), x ↦ x² | 否 | 是 | 到达域裁齐成像；可 ±1 仍共用靶点 |
| ④ | [0,∞) → [0,∞), x ↦ x² | 是 | 是 | 两病齐除：双射 |

**同一条规则，四个不同的函数，四种身份。** 若你还记得上一节「函数 = 定义域 + 到达域 + 逐点的值，三件套」，这张表就是那句话最响亮的回声：定义域与到达域不是函数的包装纸，是它的骨肉。

> 表里有句话先按下没问：④ 凭什么是满射？任给 b ≥ 0，得找 a ≥ 0 使 a² = b——也就是说**每个非负实数都得有平方根**。这件事看着天经地义，实则深不见底：它依赖 ℝ「没有缝隙」的完备性，是分析那一支的核心地基。此处暂且承认，严格证明留待实数完备性。你看，连 √2 的存在，也要等一条定理来确立——这正是我们整套从头重建的意义所在。

## 双射

f : A → B 既单又满，就叫**双射**(bijective)（同义：**一一对应**(one-to-one correspondence)）。把两个定义并排读出它的画像：B 里每个元素**至少**被打中一次（满射）、**至多**被打中一次（单射）——所以**恰好一次**。A 的元素与 B 的元素经由 f 两两牵手、无一落单、无一脚踏两船：完美配对。又见「至少 + 至多 = 恰好」——和定义函数时的 ∃! 是同一个句式，只是这回盘问的方向反了过来：那时问「每个输入恰好几支箭**射出**」，现在问「每个靶点恰好几支箭**射入**」。

## 标准证明套路，跑一遍全程

拿 f : ℝ → ℝ, x ↦ 2x + 1 当靶子（ℝ 及其四则照例先借用），把两种证明的标准姿势各演一遍。

**单射**：设 f(x) = f(x′)，即 2x + 1 = 2x′ + 1。两边减 1 得 2x = 2x′，两边除以 2 得 x = x′。∎ （从等式出发、代数消去——逆否形式的好处全在这几步里。）

**满射**：任给 b ∈ ℝ，要找 a 使 2a + 1 = b。解出候选 a = (b−1)/2——它是个实数，资格合法；代回验证：f(a) = 2·(b−1)/2 + 1 = (b−1) + 1 = b。∎ （先解出见证、再代回验证，两步缺一不可：解出来的东西必须确认真在定义域里、真打中 b。）

故 f 是双射。

## 否定形式

实战里「证明 f **不是**单射/满射」与证明「是」一样常见，所以把两个定义的否定推到底（正好用上[[fond_quantificatori|量词]]那一节的否定规则——∀∃ 对调、谓词取反）：

**非单射**：¬∀x,x′ ( f(x)=f(x′) ⟹ x=x′ ) ≡ ∃x,x′ ( f(x)=f(x′) ∧ x≠x′ )。（蕴含的否定 = 前件真而后件假——回想 p⟹q ≡ ¬p∨q，取反得 p∧¬q。）一句话：**找一对撞车的见证**。

**非满射**：¬∀b∃a f(a)=b ≡ ∃b ∀a f(a)≠b。一句话：**找一个无人能打中的靶点**——注意这后半截是个 ∀，光说「我没找到谁打中它」不够，得**论证所有** a 都打不中。

两条否定都把「证伪」变成了找见证的活——但难度不对称：非单射只需亮出两个具体的 x；非满射的见证 b 后面还拖着一条要论证的 ∀。下面练习里你会亲身体会这条不对称。

## 一道激活练习

用上面的否定形式，证明 ① 号函数 f : ℝ → ℝ, x ↦ x² **既非单射、也非满射**。要求：每条都先写出要证的量词语句，再给出见证并完成论证。

（提示：非满射那条，选定靶点 b 后，剩下的「∀a, a² ≠ b」是个全称命题——它需要一个对**所有**实数成立的理由，想想平方有什么对所有实数都成立的性质。）

## 参考解答

（建议先自己写完再看。）

**非单射**：要证 ∃x,x′ ( x² = x′² ∧ x ≠ x′ )。取见证 x = 1, x′ = −1：1² = 1 = (−1)²，而 1 ≠ −1。两个条件都验毕。∎

**非满射**：要证 ∃b ∀a ( a² ≠ b )。取见证 b = −1，往证 ∀a ∈ ℝ, a² ≠ −1。这一步不能靠「试了几个都不行」，要一个普适的理由：任何实数的平方非负（a ≥ 0 与 a < 0 两种情形下 a·a 都 ≥ 0——此处暂且承认这条关于 ℝ 序与乘法的事实，它的严格出处是有序域的公理，留待数系构造），故 a² ≥ 0 > −1，从而 a² ≠ −1，对一切 a 成立。∎

回味两口：其一，两个证明的「劳动量」印证了正文里说的不对称——非单射亮出两个数就收工，非满射在给出见证之后，还需要完成一个全称命题的论证。其二，留意「平方非负」也未被当作显然——它将由有序域的公理严格给出。这套体系里没有「显然」，只有「已证」和「待证」。

## 往前看一步

双射给了我们完美配对。配对天然让人想做一件事：**沿着箭头走回去**——既然 B 的每个元素恰好被一支箭打中，那「从 b 回到打中它的那个 a」本身岂不也是一个函数？这个直觉是对的，但要把它说严格，先得有「把两个函数接力起来」的操作。函数的复合、以及「双射恰好就是可逆的函数」这条定理，是接下来 [[fond_composizione|复合与逆函数]] 的正题。
