## 为什么需要量词

上一节我们把「命题」定得很严：一句话必须能干净地落进真或假。也正因如此，我们当时把「x > 3」这类句子**挡在了门外**——它的真假悬而未决，要看 x 是谁。可偏偏，数学里最常说的话恰恰长这样：「**对每一个**实数 x，都有 x² ≥ 0」「**存在**一个 n，使得 2ⁿ > 1000」。这些句子扫过一整个范围，是定理与定义的标准句式。这一节就把它们正式收进我们严密的语言里——办法分两步：先承认带未定元的「开句」，再用两个**量词**把它们封成真正的命题。

## 谓词

**谓词**(predicate)，也叫**开语句**(open sentence)，是一句含有**自由变元**(free variable)的话，记作 P(x)；它本身**没有**真值，但一旦把 x 代成某个具体对象，就变回一个命题。

- 取 P(x)：「x > 3」，变元 x 取遍实数。那么 P(5) 为真、P(2) 为假；而 P(x) 孤零零地写着，无所谓真假。
- 谓词可以带多个变元：Q(x, y)：「x < y」，代入 Q(1, 2) 真、Q(2, 1) 假。

有一点必须讲清：写谓词，**一定要交代变元的取值范围**——这个范围叫**论域**(domain of discourse)。「x > 3」在论域为 ℝ 时，和在论域为「正整数」时，谈的是不同的事。（这个「范围」严格来说是一个集合；集合的严格定义我们下节再给，这里先借用「一堆对象凑成一个整体」的朴素理解。）

## 两个量词

要把开句 P(x) 封成一个有真假的命题，就给变元**加上量词**、把它「绑定」住。要紧的只有两个。

**全称量词**(universal quantifier) ∀。写 ∀x ∈ A, P(x)，读作「对论域 A 里的**每一个** x，P(x) 成立」。它为真，当且仅当 P(x) 对 A 中**所有** x 都为真。

- ∀x ∈ ℝ, x² ≥ 0：真。
- ∀x ∈ ℝ, x > 3：假——因为有 x = 0 不满足。

记住要害：**要推翻一个全称命题，只需举出一个反例**(counterexample)。哪怕千万个 x 都对，只要逮着一个不对的，∀ 就塌了。

**存在量词**(existential quantifier) ∃。写 ∃x ∈ A, P(x)，读作「在 A 里**存在**某个 x，使 P(x) 成立」。它为真，当且仅当 P(x) 对 A 中**至少一个** x 为真。

- ∃x ∈ ℝ, x > 3：真——拿出 x = 5 作见证即可。
- ∃x ∈ ℝ, x² < 0：假——没有任何实数的平方是负的。

它的要害正相反：**要证一个存在命题，给出一个见证**(witness)就够了；**要推翻它，则得说明无一可行**。

## 量词的否定——德摩根律的「放大版」

这是全节最要紧、日后用得最多的一处。先建立直觉：把 ∀x P(x) 想成「对论域里每个对象都成立」——论域有限时，它无非是一长串合取 P(a) ∧ P(b) ∧ ⋯；把 ∃x P(x) 想成「至少一个成立」，有限时就是一长串析取 P(a) ∨ P(b) ∨ ⋯。

于是上一节的 [[fond_proposizioni|德摩根律]] 立刻「放大」成量词的否定规则：

- ¬(∀x P(x)) ≡ ∃x ¬P(x)。「并非人人都 P」=「存在一个不 P 的」。（对应 ¬(p ∧ q) ≡ ¬p ∨ ¬q。）
- ¬(∃x P(x)) ≡ ∀x ¬P(x)。「并不存在 P 的」=「人人都不 P」。（对应 ¬(p ∨ q) ≡ ¬p ∧ ¬q。）

口诀就一句：**否定往里钻，∀ 与 ∃ 对调，谓词随之取反**。注意上面那个「一串 ∧ / 一串 ∨」只是帮你看清来路的类比——论域无限时 ∀、∃ 并不真是有限的合取析取；但这两条否定规则对**任意**论域（含无限）都照样成立，它们直接来自 ∀、∃ 的定义。

举一例。「∀x ∈ ℝ, x² ≥ 0」的否定是「∃x ∈ ℝ, x² < 0」（把 ∀ 换成 ∃、把 x² ≥ 0 取反成 x² < 0）。原命题真，故其否定假——自洽。

## 嵌套：顺序要人命

把量词叠起来用时，**∀ 和 ∃ 的先后不能随便换**。看一个论域取 ℝ、谓词 Q(x, y) 为「y > x」的例子：

- ∀x ∃y (y > x)：「对每个 x，都**找得到**一个更大的 y」。**真**——给定 x，取 y = x + 1 即可（y 可以随 x 变）。
- ∃y ∀x (y > x)：「**存在某个** y，它比**每个** x 都大」。**假**——那等于说实数里有个最大的数，没有。

同样的两个量词、同样的谓词，仅仅调了次序，真假就翻了。症结在于：∀x ∃y 里，y 的选取**允许依赖** x；而 ∃y ∀x 里，必须**一个 y 通吃所有** x。（顺带：同类量词之间可以随意交换——∀x∀y ≡ ∀y∀x，∃x∃y ≡ ∃y∃x；只有 ∀ 与 ∃ 混着叠时才挑顺序。）这个「顺序决定一切」的现象，等你将来学极限、连续的 ε-δ 定义时会反复撞见——那些定义正是几个 ∀、∃ 叠出来的，叠错顺序，意思就全变了。

## 几个最容易栽跟头的地方

- **否定时只做一半**。把 ∀ 的否定写成「∀x ¬P(x)」（只取反谓词、忘了换量词），或写成「∃x P(x)」（只换量词、忘了取反谓词），都是错的。两件事要一起做。
- **混合量词的顺序**。如上例，调了次序就可能变味，务必照原句的先后来读。
- **空论域的边界情形**。若论域为空，∀x P(x) **恒真**（举不出反例——这正是上一节「空洞为真」的体现），而 ∃x P(x) **恒假**（找不到对象作见证）。「所有住在这间空屋里的人都两米高」是一句真话。
- **「=」暂且朴素地用**。下面练习里会出现「相等」，我们先按「指同一个对象」的朴素意思用它；它的严格身份留待讲关系时再交代。

## 一道激活练习

把下面这句的否定，**一步步把 ¬ 推到紧贴谓词为止**（最终式子里 ¬ 前面不许再有量词），写出最终形式，再用一句中文说清它断言什么、是真是假：

¬( ∀x ∈ ℝ, ∃y ∈ ℝ, y > x )。

（提示：从最外层的 ∀ 开始，一层层套用否定规则；每钻进一层，量词对调一次，最后对谓词「y > x」取反。）

## 参考解答

（建议先自己写完再看。）思路：从最外层量词起，每剥一层就套一次否定规则（∀↔∃ 对调、谓词取反），一路推到 ¬ 紧贴谓词。

```
¬( ∀x∈ℝ, ∃y∈ℝ, y>x )
≡ ∃x∈ℝ, ¬( ∃y∈ℝ, y>x )     （¬∀ → ∃¬）
≡ ∃x∈ℝ, ∀y∈ℝ, ¬(y>x)        （¬∃ → ∀¬）
≡ ∃x∈ℝ, ∀y∈ℝ, y ≤ x          （「y>x」取反为「y≤x」）
```

它断言什么：**存在一个实数 x，比每个实数 y 都不小**——也就是「实数里有个最大的数」。**是假的**：对任何 x，x+1 就比它大，没有最大实数。（这与原句「对每个 x 都找得到更大的 y」为真恰好相反，自洽。）

## 往前看一步

现在，配上上一节的联结词，我们已经能把几乎任何一句数学断言写得滴水不漏。但有一样东西，这一节自始至终是**借来用**的：量词扫过的那个「论域」——我们一直说「x 取遍 ℝ」「取遍某个范围」，可这个**范围**本身，严格来说究竟是什么？它就是**集合**：把一些对象当成一个整体来看待，好让量词有个确切的地方去「取遍」。这正是 [[fond_insiemi|集合]] 要替我们钉死的第一件事。
